## 1. Television 클래스

#### 핵심 코드

```python
class Television:
    serial_number = 0 # 클래스 변수

    def __init__(self, channel, volume, on):
        Television.serial_number += 1
        self.serial_number = Television.serial_number
        self.channel = channel
        self.volume = volume
        self.on = on
```

#### 해설

`Television`은 TV 객체를 만들기 위한 설계도이다.

- `serial_number`: 모든 TV 객체가 공유하는 클래스 변수
- `self.serial_number`: 각 객체가 가지는 고유 번호
- `self.channel`: 각 TV의 채널
- `self.volume`: 각 TV의 볼륨
- `self.on`: 각 TV의 전원 상태

객체를 만들 때마다 `Television.serial_number += 1`이 실행되므로 TV 객체 번호가 자동으로 증가한다.

## 3. 메소드와 self

```python
def set_channel(self, channel):
    self.channel = channel

def get_channel(self):
    return self.channel
```

클래스 안에 정의된 함수를 메소드라고 한다.

- `set_channel()`은 채널 값을 변경한다.
- `get_channel()`은 현재 채널 값을 반환한다.
- `self`는 이 메소드를 호출한 객체 자신이다.

```python
print(tv1.get_channel())
print(tv1.channel)
```

둘 다 `tv1`의 채널 값을 확인하는 코드이다.

### 5. Circle 클래스

#### 핵심 코드

```python
import math

class Circle:
    def __init__(self, radius = 0):
        self.radius = radius

    def getArea(self):
        return math.pi * (self.radius ** 2)

    def getPerimeter(self):
        return 2 * math.pi * self.radius
```

#### 해설

`Circle` 클래스는 원을 객체로 표현한다.

- 속성: `radius`
- 동작: `getArea()`, `getPerimeter()`

```python
c1 = Circle(10)
print(c1.getArea())
print(c1.getPerimeter())
```

반지름이 10인 원의 넓이와 둘레를 계산한다.

### 7. Rectangle 클래스와 `__add__`

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def get_area(self):
        return self.width * self.height

    def __str__(self):
        return f"Rectangle(width={self.width}, height={self.height})"

    def __add__(self, other):
        new_width = self.width + other.width
        new_height = self.height + other.height
        return Rectangle(new_width, new_height)
```

#### 핵심 포인트

`__add__`를 정의했기 때문에 객체끼리 `+` 연산을 할 수 있다.

```python
rect1 = Rectangle(10, 20)
rect2 = Rectangle(30, 40)
rect3 = rect1 + rect2
print(rect3)
```

실제로는 아래 코드가 실행되는 느낌이다.

```python
rect3 = rect1.__add__(rect2)
```

결과:

```text
Rectangle(width=40, height=60)
```

### 8. Dice 클래스

```python
from random import randint

class Dice:
    def __init__(self, x, y):
        self._x = x
        self._y = y
        self._size = 30
        self.value = 1

    def roll_dice(self):
        self.value = randint(1, 6)

    def print_dice(self):
        print(f"Dice value: {self.value}")
```

주사위 객체를 만들고 `roll_dice()`를 호출하면 1부터 6 사이의 값으로 바뀐다.

```python

d.print_dice()
```

### 10. 다형성: Shape, Circle, Rectangle

```python
class Shape:
    def __init__(self, name):
```

`Shape`는 공통 부모 클래스이다.

같은 `area()` 호출이지만 실제 객체가 `Circle`인지 `Rectangle`인지에 따라 다르게 동작한다.

## 4. 파일 열기

```python
def open_file(self):
    if self._should_save():
        path = filedialog.askopenfilename(
            defaultextension=".txt",
            filetypes=[("텍스트 파일", "*.txt"), ("모든 파일", "*.*")]
        )
        if path:
            with open(path, "r", encoding="utf-8") as f:
                self.text.delete("1.0", tk.END)
                self.text.insert("1.0", f.read())
```

#### 흐름

```text
1. 현재 문서에 저장할 변경사항이 있는지 확인
2. 파일 선택 창 띄우기
3. 사용자가 파일을 고르면 open()으로 읽기
4. 기존 Text 내용을 삭제
5. 읽은 내용을 Text 위젯에 삽입
```

`"1.0"`은 Text 위젯의 첫 번째 줄, 0번째 글자 위치를 의미한다.  
`tk.END`는 마지막 위치를 의미한다.

### 5. 파일 저장

```python
def save_file(self):
    if self.file_path:
        self._write_file(self.file_path)
        return True
    return self.save_as()
```

이미 파일 경로가 있으면 그 경로에 저장하고, 없으면 다른 이름으로 저장을 실행한다.

실제 파일 쓰기:

```python
def _write_file(self, path):
    with open(path, "w", encoding="utf-8") as f:
        f.write(self.text.get("1.0", tk.END))
    self.text.edit_modified(False)
```

`Text` 위젯의 전체 내용을 가져와 파일에 쓴다.

### 6. 저장 여부 확인

```python
def _should_save(self):
    if self.text.edit_modified():
        answer = messagebox.askyesnocancel("저장", "변경 내용을 저장하시겠습니까?")
        if answer is None:
            return False
        if answer:
            return self.save_file()
        return True
    return True
```

창을 닫거나 새 파일을 열기 전에 수정된 내용이 있으면 저장 여부를 묻는다.

|사용자 선택|동작|
|---|---|
|예|저장 후 진행|
|아니오|저장하지 않고 진행|
|취소|작업 취소|

|`<B1-Motion>`|왼쪽 버튼을 누른 채 이동|`draw()`|
|`<ButtonRelease-1>`|왼쪽 버튼 뗌|`stop_draw()`|

GUI 프로그램은 사용자의 행동이 이벤트가 되고, 이벤트가 콜백 함수를 실행한다.

---

### 11. 선 그리기

```python
def draw(self, event):
    if self.last_x is None:
        return

    self.canvas.create_line(
        self.last_x, self.last_y, event.x, event.y,
        fill=self.current_color(),
        width=self.width,
        capstyle="round",
        smooth=True
    )

    self.last_x = event.x
    self.last_y = event.y
```

이전 좌표에서 현재 좌표까지 선을 긋고, 현재 좌표를 다시 이전 좌표로 저장한다.  
이 과정을 마우스가 움직일 때마다 반복하면 연속된 선처럼 보인다.

---

### 12. 지우개 구현

```python
def current_color(self):
    return "white" if self.eraser else self.color
```

지우개는 실제로 선을 삭제하는 방식이 아니라, 배경색인 흰색으로 다시 그리는 방식이다.

---

### 13. NumPy 배열 변환

```python
import numpy as np

ftemp = [32, 63, 73, 82, 91, 100, 110, 120, 80, 140]
F = np.array(ftemp)

print(ftemp)
print(F)
```

Python 리스트:

```text
[32, 63, 73, 82, 91, 100, 110, 120, 80, 140]
```

NumPy 배열:

```text
[ 32  63  73  82  91 100 110 120  80 140]
```

리스트와 배열은 비슷해 보이지만 목적이 다르다.

|구분|Python 리스트|NumPy 배열|
|---|---|---|
|목적|일반 데이터 보관|수치 계산|
|계산 방식|반복문 중심|배열 연산 중심|
|AI/데이터 분석|가능하지만 불편|매우 자주 사용|

### 6. 파일 저장

### 7. 선 그리기

```python
def draw(self, event):
    if self.last_x is None:
        return

    self.canvas.create_line(
        self.last_x, self.last_y,
        event.x, event.y,
        fill=self.current_color(),
        width=self.width,
        capstyle="round",
        smooth=True
    )

    self.last_x = event.x
    self.last_y = event.y
```

이전 마우스 좌표에서 현재 마우스 좌표까지 선을 긋는다.

이 과정을 계속 반복하면 사용자가 드래그한 경로가 선으로 보인다.
