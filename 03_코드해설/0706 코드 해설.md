---

### 핵심 코드 1: Television 클래스

```python
class Television:
    serial_number = 0

    def __init__(self, channel, volume, on):
        Television.serial_number += 1
        self.serial_number = Television.serial_number
        self.channel = channel
        self.volume = volume
        self.on = on

    def __str__(self):
        return f"Television(channel={self.channel}, volume={self.volume}, on={self.on})"
```

### 해설

`Television`은 TV 객체를 만들기 위한 설계도이다.

- `serial_number`: 모든 객체가 공유하는 클래스 변수
- `self.channel`: 객체마다 따로 가지는 인스턴스 변수
- `__init__`: 객체가 생성될 때 실행되는 생성자
- `__str__`: 객체를 출력할 때 사용할 문자열 정의

```python
tv1 = Television(1, 10, True)
tv2 = Television(2, 20, False)
```

같은 클래스로 만든 객체라도 서로 다른 채널, 볼륨, 전원 상태를 가질 수 있다.

---

### 핵심 코드 2: Circle 클래스

```python
class Circle:
    def __init__(self, radius=0):
        self.radius = radius

    def getArea(self):
        return math.pi * (self.radius ** 2)

    def getPerimeter(self):
        return 2 * math.pi * self.radius
```

### 해설

원 객체를 만들고, 원의 넓이와 둘레를 메소드로 계산한다.

```text
속성 = radius
동작 = getArea(), getPerimeter()
```

---

### 핵심 코드 3: Rectangle과 `__add__`

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def __add__(self, other):
        new_width = self.width + other.width
        new_height = self.height + other.height
        return Rectangle(new_width, new_height)
```

### 해설

`__add__`를 정의하면 객체끼리 `+` 연산을 할 수 있다.

```python
rect3 = rect1 + rect2
```

실제로는 `rect1.__add__(rect2)`가 실행되는 느낌이다.

### 핵심 코드 4: 상속과 다형성

```python
class ElectricCar(Car):
    def __init__(self, make, model, color, price, battery_capacity):
        super().__init__(make, model, color, price)
        self.battery_capacity = battery_capacity
```

전기차는 자동차의 속성을 물려받고, 배터리 용량을 추가한다.

```python
for shape in shapeList:
    print(shape.area())
```

같은 `area()` 호출이라도 객체가 원이면 원의 넓이, 사각형이면 사각형의 넓이를 계산한다. 이것이 다형성이다.

## 3. `0706/2.ipynb` tkinter GUI와 NumPy

### 이 파일의 목적

`tkinter`로 메모장과 그림판을 만들고, 마지막에 NumPy 배열 변환을 실습한 노트북이다.

```text
메모장 GUI
→ 그림판 GUI
→ NumPy 배열 변환
```

---

### 메모장 핵심 구조

```python
class Notepad:
    def __init__(self, root):
        self.root = root
        self.root.title("메모장")
        self.root.geometry("800x600")
        self.file_path = None
        self.create_widgets()
```

메모장 전체를 `Notepad` 클래스로 묶었다.

- `root`: 메인 창
- `file_path`: 현재 파일 경로
- `create_widgets()`: 텍스트창, 메뉴, 스크롤바 생성

---

### Text 위젯

```python
self.text = tk.Text(self.root, undo=True, wrap="word")
self.text.pack(fill="both", expand=True)
```

`Text`는 여러 줄 글을 입력할 수 있는 위젯이다.

---

### 파일 열기/저장

```python
with open(path, "r", encoding="utf-8") as f:
    self.text.delete("1.0", tk.END)
    self.text.insert("1.0", f.read())
```

```python
with open(path, "w", encoding="utf-8") as f:
    f.write(self.text.get("1.0", tk.END))
```

`"1.0"`은 Text 위젯의 시작 위치이고, `tk.END`는 끝 위치이다.

---

### 그림판 핵심 구조

```python
self.canvas = tk.Canvas(master, bg="white", highlightthickness=0)
self.canvas.pack(fill="both", expand=True)
```

`Canvas`는 그림을 그리는 영역이다.

```python
self.canvas.bind("<Button-1>", self.start_draw)
self.canvas.bind("<B1-Motion>", self.draw)
self.canvas.bind("<ButtonRelease-1>", self.stop_draw)
```

마우스를 누르고, 움직이고, 떼는 행동을 각각 함수와 연결했다.

---

### 선 그리기

```python
self.canvas.create_line(
    self.last_x, self.last_y,
    event.x, event.y,
    fill=self.current_color(),
    width=self.width,
    capstyle="round",
    smooth=True
)
```

이전 좌표와 현재 좌표를 선으로 연결한다. 이 동작이 반복되면 사용자가 드래그한 경로가 그림으로 남는다.

---

### NumPy 배열 변환

```python
import numpy as np

ftemp = [32, 63, 73, 82, 91]
F = np.array(ftemp)
```

Python 리스트를 NumPy 배열로 변환했다. 나중에 데이터 분석, 이미지 처리, AI 입력값을 다룰 때 이런 배열 구조가 자주 사용된다.
