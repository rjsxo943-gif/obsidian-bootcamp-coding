# 1. `stringEx.py` 코드 해설

## 원본 코드 정리

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

# print(f"You entered: {a} and {b}")

print(f"a + b = {a + b}")
print(f"a - b = {a - b}")
print(f"a * b = {a * b}")
print(f"a / b = {a / b}")
```

---

## 코드 목적

사용자에게 숫자 2개를 입력받고, 두 숫자에 대해 사칙연산 결과를 출력하는 코드이다.

---

## 코드 흐름

```text
1. 첫 번째 숫자 입력
2. 두 번째 숫자 입력
3. 입력값을 정수로 변환
4. 덧셈, 뺄셈, 곱셈, 나눗셈 수행
5. f-string으로 결과 출력
```

---

## 핵심 코드 1: 입력과 형변환

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
```

`input()`은 사용자에게 값을 입력받는 함수이다.

하지만 `input()`으로 받은 값은 기본적으로 문자열이다.

예를 들어 사용자가 `10`을 입력해도 Python 내부에서는 숫자 `10`이 아니라 문자열 `"10"`으로 저장된다.

그래서 숫자 계산을 하려면 `int()`로 정수 변환을 해야 한다.

```python
int(input("Enter first number: "))
```

---

## 핵심 코드 2: f-string 출력

```python
print(f"a + b = {a + b}")
```

`f""` 안에서 `{}`를 사용하면 변수나 계산식을 문자열 안에 바로 넣을 수 있다.

예를 들어 `a = 10`, `b = 3`이면 다음처럼 출력된다.

```text
a + b = 13
```

---

## 사칙연산 정리

|코드|의미|
|---|---|
|`a + b`|덧셈|
|`a - b`|뺄셈|
|`a * b`|곱셈|
|`a / b`|나눗셈|

---

## 주의할 점: 0으로 나누기

나눗셈에서 `b`가 0이면 오류가 발생한다.

```python
a / b
```

만약 `b = 0`이면 다음 오류가 발생한다.

```text
ZeroDivisionError
```

조건문을 배우면 아래처럼 오류를 막을 수 있다.

```python
if b == 0:
    print("0으로 나눌 수 없습니다.")
else:
    print(a / b)
```

---

## 개선해서 다시 쓰면

```python
a = int(input("Enter first number: "))

print(f"a + b = {a + b}")

print(f"a * b = {a * b}")

if b == 0:
    print("0으로 나눌 수 없습니다.")
else:
    print(f"a / b = {a / b}")
```

이렇게 하면 사용자가 두 번째 숫자로 0을 입력해도 프로그램이 바로 오류로 종료되지 않는다.

# 2. `1.py` 코드 해설

## 원본 코드 정리

```python
y = []
i = 0
c = 0

for i in range(5):
    a = int(input("5개의 정수값을 입력하시오: "))
    y.append(a)

c = int(len(y))

print(y)
print(f"mean = {sum(y) / c}, sum = {sum(y)}, min = {min(y)}, max = {max(y)}")
```

---

## 코드 목적

정수 5개를 입력받아 리스트에 저장하고, 평균, 합계, 최솟값, 최댓값을 계산하는 코드이다.

---

## 코드 흐름

```text
1. 빈 리스트 생성
2. for문으로 5번 반복
3. 사용자에게 정수 입력받기
4. 입력값을 리스트에 추가
5. 리스트 전체 출력
6. 합계, 평균, 최솟값, 최댓값 출력
```

---

## 핵심 코드 1: 빈 리스트 생성

```python
y = []
```

`[]`는 빈 리스트를 만든다.

리스트는 여러 값을 순서대로 저장할 수 있는 자료구조이다.

---

## 핵심 코드 2: 반복 입력

```python
for i in range(5):
    a = int(input("5개의 정수값을 입력하시오: "))
    y.append(a)
```

`range(5)`는 0부터 4까지 총 5개의 값을 만든다.

즉, `for`문이 5번 반복된다.

반복할 때마다 숫자를 입력받고, `append()`로 리스트에 추가한다.

---

## 핵심 코드 3: 리스트에 값 추가

```python
y.append(a)
```

`append()`는 리스트의 맨 뒤에 값을 추가한다.

예를 들어 입력값이 `10`, `20`, `30`, `40`, `50`이면 리스트는 다음처럼 된다.

```python
[10, 20, 30, 40, 50]
```

---

## 핵심 코드 4: 리스트 통계

```python
sum(y)
len(y)
min(y)
max(y)
```

|함수|의미|
|---|---|
|`sum(y)`|리스트 요소의 합계|
|`len(y)`|리스트 요소 개수|
|`min(y)`|최솟값|
|`max(y)`|최댓값|

평균은 다음 공식으로 계산한다.

```python
sum(y) / len(y)
```

---

## 개선 포인트

원본 코드에서는 `i = 0`, `c = 0`을 먼저 선언했다.

하지만 이 두 줄은 없어도 된다.

```python
i = 0
c = 0
```

또한 `len(y)`는 이미 정수이므로 `int(len(y))`처럼 다시 정수 변환을 하지 않아도 된다.

```python
c = len(y)
```

---

## 개선해서 다시 쓰면

```python
scores = []

for i in range(5):
    score = int(input("5개의 정수값을 입력하시오: "))
    scores.append(score)

count = len(scores)

print(scores)
print(f"mean = {sum(scores) / count}, sum = {sum(scores)}, min = {min(scores)}, max = {max(scores)}")
```

변수 이름도 `y`보다 `scores`처럼 의미가 있는 이름을 쓰면 코드 이해가 쉬워진다.

# 3. `2.py` 코드 해설

## 원본 코드 정리

```python
fruits = []

for i in range(5):
    fruit = input("enter a fruit: ")
    fruits.append(fruit)

print(fruits)
print(fruits[-1])
print(fruits[3])
```

---

## 코드 목적

과일 이름 5개를 입력받아 리스트에 저장하고, 리스트 전체와 특정 위치의 값을 출력하는 코드이다.

---

## 코드 흐름

```text
1. 빈 리스트 생성
2. 과일 이름 5개 입력
3. 리스트에 과일 이름 저장
4. 리스트 전체 출력
5. 마지막 요소 출력
6. 네 번째 요소 출력
```

---

## 핵심 코드 1: 문자열 리스트 입력

```python
fruit = input("enter a fruit: ")
fruits.append(fruit)
```

`input()`으로 입력받은 과일 이름은 문자열이다.

예를 들어 다음처럼 입력했다고 하자.

```text
apple
banana
melon
grape
orange
```

리스트는 다음처럼 저장된다.

```python
["apple", "banana", "melon", "grape", "orange"]
```

---

## 핵심 코드 2: 마지막 요소 출력

```python
print(fruits[-1])
```

`-1` 인덱스는 리스트의 마지막 요소를 의미한다.

```python
fruits[-1]
```

위 예시에서는 `"orange"`가 출력된다.

---

## 핵심 코드 3: 특정 인덱스 출력

```python
print(fruits[3])
```

Python 리스트의 인덱스는 0부터 시작한다.

|인덱스|의미|예시 값|
|---|---|---|
|`0`|첫 번째 요소|`"apple"`|
|`1`|두 번째 요소|`"banana"`|
|`2`|세 번째 요소|`"melon"`|
|`3`|네 번째 요소|`"grape"`|
|`-1`|마지막 요소|`"orange"`|

따라서 `fruits[3]`은 네 번째 과일을 의미한다.

---

## 주의할 점: 인덱스 오류

```text
IndexError
```

## 4-1. 리스트 정렬

```python
numbers = [5, 2, 8, 1]

result1 = sorted(numbers)
result2 = sorted(numbers, reverse=True)

print(result1)
print(result2)
print(numbers)
```

---

## 코드 해석

`sorted(numbers)`는 정렬된 새 리스트를 반환한다.

```python
result1 = sorted(numbers)
```

결과:

```python
[1, 2, 5, 8]
```

`reverse=True`를 주면 내림차순으로 정렬한다.

```python
result2 = sorted(numbers, reverse=True)
```

결과:

```python
[8, 5, 2, 1]
```

중요한 점은 `sorted()`는 원본 리스트를 바꾸지 않는다는 것이다.

```python
print(numbers)
```

원본은 그대로 다음과 같다.

```python
[5, 2, 8, 1]
```

---

## `sorted()`와 `sort()` 차이

|코드|의미|원본 변경|
|---|---|---|
|`sorted(numbers)`|정렬된 새 리스트 반환|변경 안 함|
|`numbers.sort()`|원본 리스트 자체를 정렬|변경함|

## 4-2. 튜플

```python
t1 = ()
t2 = (1,)
t3 = (1, 2, 3)
t4 = 1, 2, 3
t5 = ("a", "b", ("ab", "cd"))
```

---

## 코드 해석

튜플은 리스트와 비슷하지만 값을 변경할 수 없다.

리스트는 대괄호 `[]`를 사용하고, 튜플은 보통 소괄호 `()`를 사용한다.

```python
numbers = [1, 2, 3]      # 리스트
numbers_tuple = (1, 2, 3) # 튜플
```

---

## 요소가 하나인 튜플

요소가 하나인 튜플은 반드시 쉼표가 필요하다.

```python
t2 = (1,)
```

쉼표가 없으면 튜플이 아니라 그냥 값으로 인식된다.

```python
t = (1)
```

이 경우 `t`는 튜플이 아니라 정수 `1`이다.

```python
student_info = {
```

## 4-5. 딕셔너리에 값 추가

```python
a = {1: "a"}
a[2] = "b"
```

## 코드 해석

## 4-7. 조건문 실습

```python
money = int(input("돈얼마있노: "))

if money >= 20000:
    print("돈많노 texi")
else:
    print("ㅠㅠ walk")

print("Good night")
```

---

## 코드 해석

이 코드는 돈이 20000원 이상이면 택시를 타고, 아니면 걸어간다는 조건문이다.

핵심 구조는 다음과 같다.

```python
if 조건:
    조건이 참일 때 실행
else:
    조건이 거짓일 때 실행
```

`money >= 20000`이 참이면 첫 번째 출력문이 실행되고, 거짓이면 `else` 아래의 출력문이 실행된다.

## 4-8. 학점 판별

```python
score = int(input("점수: "))

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
elif score >= 60:
    print("D")
else:
    print("F")
```

---

## 코드 해석

`elif`는 여러 조건을 순서대로 검사할 때 사용한다.

점수 범위에 따라 A, B, C, D, F를 출력한다.

중요한 점은 위에서부터 조건을 검사한다는 것이다.

예를 들어 `score = 85`라면:

```text
score >= 90 → 거짓
score >= 80 → 참
```

따라서 `B`가 출력된다.

## 4-9. 윤년 판별

```python
year = int(input("연도: "))

if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
    print(f"{year}년은 윤년입니다.")
else:
    print(f"{year}년은 평년입니다.")
```

---

## 코드 해석

윤년 조건은 다음과 같다.

```text
4로 나누어떨어지고 100으로는 나누어떨어지지 않으면 윤년
또는
400으로 나누어떨어지면 윤년
```

원본 조건식은 동작할 수 있지만, 괄호가 없어서 처음 볼 때 의미가 헷갈릴 수 있다.

조금 더 명확하게 쓰면 다음과 같다.

```python
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year}년은 윤년입니다.")
else:
    print(f"{year}년은 평년입니다.")
```

괄호를 넣으면 조건의 의미가 더 잘 보인다.

# ✅ 오늘의 핵심 정리

```text
input()은 문자열로 입력받는다.
숫자 계산을 하려면 int()로 형변환해야 한다.
f-string은 문자열 안에 변수나 계산식을 넣는 출력 방식이다.
리스트는 여러 값을 순서대로 저장한다.
append()는 리스트에 값을 추가한다.
인덱스는 0부터 시작한다.
-1은 마지막 요소를 의미한다.
sorted()는 정렬된 새 리스트를 반환하고 원본은 유지한다.
튜플은 변경할 수 없는 자료구조이다.
세트는 중복을 제거한다.
딕셔너리는 key-value 구조이다.
if, elif, else로 조건에 따라 실행 흐름을 나눌 수 있다.
```
