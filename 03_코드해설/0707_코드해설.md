## 1. gTTS로 음성 파일 만들기

```python
from gtts import gTTS

text = "오늘 배운 내용을 음성으로 저장합니다."
tts = gTTS(text=text, lang="ko")
tts.save("news_Son.mp3")
```

## 코드 해설

```python
from gtts import gTTS
```

`gtts` 라이브러리에서 `gTTS` 기능만 가져온다.

```python
text = "오늘 배운 내용을 음성으로 저장합니다."
```

음성으로 바꿀 문자열을 변수에 저장한다.

```python
tts = gTTS(text=text, lang="ko")
```

텍스트를 한국어 음성으로 변환할 준비를 한다.

```python
tts.save("news_Son.mp3")
```

변환된 음성을 `news_Son.mp3` 파일로 저장한다.

## 핵심 흐름

```text
문자열 준비
→ gTTS 객체 생성
→ mp3 파일 저장
```

# 8. iloc로 행과 열 선택


# 23. 최종 정리

오늘 코드는 Python의 역할이 다음처럼 확장되는 것을 보여준다.

```text
문법 연습용 언어
→ 파일 생성 도구
→ 데이터 분석 도구
→ 머신러닝 실험 도구
→ 시각화 도구
```

오늘 가장 중요한 코드는 다음 흐름이다.

```python
iris = load_iris()

df = pd.DataFrame(
    iris.data,
    columns=['sepal_length', 'sepal_width', 'petal_length', 'petal_width']
)

s = svm.SVC(gamma=0.1, C=10)
s.fit(iris.data, iris.target)

res = s.predict(new_d)

fig = px.scatter_3d(
    df,
    x='sepal_length',
    y='sepal_width',
    z='petal_width',
    color='species'
)
```

이 흐름을 이해하면 앞으로 다른 데이터셋에서도 비슷한 방식으로 머신러닝 실습을 확장할 수 있다.
