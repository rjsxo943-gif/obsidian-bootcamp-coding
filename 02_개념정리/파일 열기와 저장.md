# 파일 열기와 저장

## 핵심 요약

파일 입출력은 프로그램 안의 데이터를 실제 파일로 읽고 쓰는 과정이다.  
메모장 프로그램에서는 `filedialog`로 파일 경로를 고르고, `open()`으로 파일을 읽거나 저장했다.

---

## 파일 열기

```python
path = filedialog.askopenfilename(
    defaultextension=".txt",
    filetypes=[("텍스트 파일", "*.txt"), ("모든 파일", "*.*")]
)
```

사용자에게 파일 선택 창을 보여준다.

```python
with open(path, "r", encoding="utf-8") as f:
    data = f.read()
```

파일을 읽는다.

---

## 파일 저장

```python
with open(path, "w", encoding="utf-8") as f:
    f.write(self.text.get("1.0", tk.END))
```

텍스트 위젯의 내용을 파일에 쓴다.

---

## open() 모드

|모드|의미|
|---|---|
|`"r"`|읽기|
|`"w"`|쓰기. 기존 내용 덮어쓰기|
|`"a"`|추가하기|
|`"x"`|새 파일 만들기. 이미 있으면 오류|

---

## encoding

```python
encoding="utf-8"
```

한글이 깨지지 않게 하기 위해 중요하다.

---

## Text 위젯 범위

```python
self.text.get("1.0", tk.END)
```

|표현|의미|
|---|---|
|`"1.0"`|1번째 줄 0번째 글자|
|`tk.END`|마지막 위치|

즉 텍스트창 전체 내용을 가져온다는 뜻이다.

---

## 내 식으로 이해

파일 저장은 결국 이거다.

```text
화면에 있는 글자 가져오기 → 파일 열기 → 파일에 쓰기
```

파일 열기는 반대다.

```text
파일 고르기 → 파일 읽기 → 화면에 넣기
```
