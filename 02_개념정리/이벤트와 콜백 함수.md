# 이벤트와 콜백 함수

## 핵심 요약

이벤트는 사용자의 행동이고, 콜백 함수는 그 행동이 발생했을 때 실행되는 함수이다.

```text
이벤트 발생 → 연결된 함수 실행
```

---

## 버튼 이벤트

```python
tk.Button(toolbar, text="색상", command=self.choose_color)
```

버튼을 클릭하면 `choose_color()`가 실행된다.

여기서 `command=self.choose_color`가 콜백 연결이다.

---

## 키보드 이벤트

```python
self.root.bind_all("<Control-s>", lambda e: self.save_file())
```

`Ctrl + S`를 누르면 `save_file()`이 실행된다.

---

## 마우스 이벤트

```python
self.canvas.bind("<Button-1>", self.start_draw)
self.canvas.bind("<B1-Motion>", self.draw)
self.canvas.bind("<ButtonRelease-1>", self.stop_draw)
```

|이벤트|의미|
|---|---|
|`<Button-1>`|마우스 왼쪽 버튼 누름|
|`<B1-Motion>`|왼쪽 버튼을 누른 채 이동|
|`<ButtonRelease-1>`|왼쪽 버튼에서 손을 뗌|

---

## event 객체

```python
def draw(self, event):
    print(event.x, event.y)
```

마우스 이벤트가 발생하면 `event` 객체 안에 좌표 정보가 들어온다.

- `event.x`: 마우스 x좌표
- `event.y`: 마우스 y좌표

---

## 내 식으로 이해

GUI는 코드를 위에서 아래로 한 번 쭉 실행하고 끝나는 프로그램이 아니다.

GUI는 켜져 있다가 사용자가 뭔가 하면 반응한다.

```text
버튼 클릭 → 함수 실행
마우스 이동 → 함수 실행
키보드 입력 → 함수 실행
```

이 흐름을 가능하게 하는 게 이벤트와 콜백 함수다.
