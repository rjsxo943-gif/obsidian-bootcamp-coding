# 인스턴스 변수와 클래스 변수

## 핵심 요약

인스턴스 변수는 객체마다 따로 가지는 값이고, 클래스 변수는 같은 클래스로 만든 객체들이 공유하는 값이다.

---

## 인스턴스 변수

```python
self.channel = channel
self.volume = volume
self.on = on
```

`self`가 붙은 변수는 보통 인스턴스 변수이다.

객체마다 값이 다를 수 있다.

```python
tv1 = Television(1, 10, True)
tv2 = Television(2, 20, False)
```

|객체|channel|volume|on|
|---|---:|---:|---|
|`tv1`|1|10|True|
|`tv2`|2|20|False|

---

## 클래스 변수

```python
class Television:
    serial_number = 0
```

클래스 바로 아래에 정의된 변수이다.  
모든 객체가 공유한다.

```python
Television.serial_number += 1
self.serial_number = Television.serial_number
```

객체가 생성될 때마다 클래스 변수 값을 증가시켜 객체 번호처럼 사용할 수 있다.

---

## 차이 정리

|구분|인스턴스 변수|클래스 변수|
|---|---|---|
|저장 위치|각 객체 안|클래스 안|
|공유 여부|객체마다 따로|모든 객체가 공유|
|예시|`self.channel`|`Television.serial_number`|
|사용 상황|객체별 상태 저장|공통 값, 누적 번호, 설정값|

---

## 내 식으로 이해

인스턴스 변수는 “각자 자기 주머니에 든 값”이다.  
클래스 변수는 “반 전체가 같이 보는 칠판 값”이다.

TV 객체마다 채널과 볼륨은 다를 수 있지만, 전체 TV가 몇 개 만들어졌는지 세는 번호는 클래스가 관리하는 게 자연스럽다.
