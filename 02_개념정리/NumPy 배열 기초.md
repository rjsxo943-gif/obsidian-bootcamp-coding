# NumPy 배열 기초

## 한 줄 정의

NumPy 배열은 숫자 데이터를 효율적으로 계산하기 위한 배열 자료구조이다.

```python
import numpy as np

ftemp = [32, 63, 73, 82, 91]
F = np.array(ftemp)
```

Python 리스트를 NumPy 배열로 변환하면 수치 계산, 데이터 분석, AI 입력 데이터 처리에 더 적합한 형태가 된다.

---

## 리스트와 NumPy 배열 차이

|구분|Python 리스트|NumPy 배열|
|---|---|---|
|목적|일반 데이터 저장|수치 계산|
|원소 타입|섞을 수 있음|보통 같은 타입|
|계산 방식|반복문 중심|배열 단위 연산 가능|
|사용 분야|일반 Python 코드|데이터 분석, 행렬 계산, AI|

---

## 출력 차이

Python 리스트는 쉼표가 보인다.

```text
[32, 63, 73, 82, 91]
```

NumPy 배열은 쉼표 없이 출력된다.

```text
[32 63 73 82 91]
```

---

## 내 식으로 이해

리스트가 일반 보관함이라면, NumPy 배열은 계산용 보관함이다.

오늘은 단순히 리스트를 배열로 바꾸는 정도였지만, 나중에 이미지, 센서 데이터, 행렬, AI 입력값도 결국 배열로 다루게 된다.
