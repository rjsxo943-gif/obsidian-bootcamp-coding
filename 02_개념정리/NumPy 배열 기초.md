# NumPy 배열 기초

## 핵심 요약

NumPy 배열은 숫자 데이터를 효율적으로 다루기 위한 자료구조이다.  
Python 리스트와 비슷해 보이지만 수치 계산에 더 적합하다.

---

## 기본 사용

```python
import numpy as np

ftemp = [32, 63, 73, 82, 91]
F = np.array(ftemp)
```

Python 리스트를 NumPy 배열로 변환한다.

---

## 출력 차이

Python 리스트:

```text
[32, 63, 73, 82, 91]
```

NumPy 배열:

```text
[32 63 73 82 91]
```

쉼표 없이 출력되는 것이 특징이다.

---

## 리스트와 배열 차이

|구분|Python 리스트|NumPy 배열|
|---|---|---|
|목적|범용 데이터 저장|수치 계산|
|원소 타입|섞을 수 있음|보통 같은 타입|
|계산|반복문 많이 사용|배열 단위 연산 가능|
|AI/데이터 분석|가능하지만 불편|표준처럼 사용|

---

## 왜 중요한가?

AI, 데이터 분석, 행렬 계산에서는 숫자 데이터가 매우 많다.  
이때 일반 리스트보다 NumPy 배열이 계산에 적합하다.

```text
리스트 = 일반 보관함
NumPy 배열 = 계산용 보관함
```

---

## 내 식으로 이해

NumPy는 Python에서 숫자 계산을 본격적으로 하기 위한 도구다.

오늘은 단순히 리스트를 배열로 바꾸는 정도였지만, 나중에는 이미지, 센서 데이터, AI 입력값도 결국 배열로 다루게 된다.
