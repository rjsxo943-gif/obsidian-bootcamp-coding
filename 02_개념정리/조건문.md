## 한 줄 정의

조건문은 조건에 따라 실행할 코드를 선택하는 문법이다.

프로그램이 항상 위에서 아래로만 실행되는 것이 아니라, 특정 조건에 따라 다른 동작을 하게 만들 수 있다.

```python
if 조건:
    실행문
else:
    실행문
```

## 핵심 개념

- 제어문
    
- 조건문
    
- `if`
    
- `else`
    
- `elif`
    
- 비교 연산자
    
- 논리 연산자
    
- 중첩 조건문
    
- 연속 조건문
    
- 조건부 표현식
    

## 1. 제어문

제어문은 프로그램의 실행 흐름을 바꾸는 문법이다.

제어문은 크게 두 가지로 나눌 수 있다.

|종류|의미|
|---|---|
|조건문|조건에 따라 실행 흐름 선택|
|반복문|특정 코드를 여러 번 반복|

조건문은 프로그램에 판단 능력을 넣는 문법이다.

## 2. `if`문

`if`는 조건이 참일 때만 코드를 실행한다.

```python
price = 30000

if price > 20000:
    print("배송비 무료")
```

`price > 20000`이 참이면 출력되고, 거짓이면 아무 일도 일어나지 않는다.

## 3. `if-else`문

조건이 참일 때와 거짓일 때의 실행을 나눌 수 있다.

```python
price = int(input("상품의 가격: "))

if price > 20000:
    shipping_cost = 0
else:
    shipping_cost = 3000

print("배송비 =", shipping_cost)
```

## 4. 들여쓰기와 블록

Python은 중괄호 `{}` 대신 들여쓰기로 코드 블록을 구분한다.

```python
if score >= 90:
    print("A")
    print("우수")
else:
    print("A 아님")
```

들여쓰기가 같은 줄끼리 같은 블록에 속한다.

조건문에서 들여쓰기를 잘못하면 코드 의미가 달라지거나 오류가 난다.

## 5. 비교 연산자

|연산자|의미|
|---|---|
|`==`|같다|
|`!=`|같지 않다|
|`>`|크다|
|`<`|작다|
|`>=`|크거나 같다|
|`<=`|작거나 같다|

예시:

```python
score = 85

if score >= 80:
    print("합격")
else:
    print("불합격")
```

## 6. Boolean 값

조건식의 결과는 `True` 또는 `False`이다.

```python
radius = 100
flag = radius > 32

print(flag)
```

결과:

```text
True
```

Boolean 변수는 조건문에 바로 사용할 수 있다.

```python
expensive = price > 20000

if expensive:
    shipping_cost = 0
else:
    shipping_cost = 3000
```

## 7. 문자열 비교

문자열도 비교할 수 있다.

```python
s1 = "apple"
s2 = "banana"

print(s1 < s2)
```

문자열 비교는 사전 순서에 가깝게 동작한다.

## 8. 실수 비교 주의

실수는 컴퓨터 내부에서 정확히 표현되지 않을 수 있다.

```python
from math import sqrt

n = sqrt(3.0)

if n * n == 3.0:
    print("같다")
else:
    print("같지 않다")
```

실수 비교는 오차 범위를 두고 비교하는 것이 좋다.

```python
if abs(n * n - 3.0) < 0.00001:
    print("거의 같다")
```

## 9. 짝수와 홀수 판별

```python
number = int(input("정수를 입력하시오: "))

if number % 2 == 0:
    print("짝수입니다.")
else:
    print("홀수입니다.")
```

`%`는 나머지 연산자이다.

2로 나눈 나머지가 0이면 짝수이다.

## 10. 논리 연산자

여러 조건을 결합할 때 사용한다.

|연산자|의미|
|---|---|
|`and`|둘 다 참이면 참|
|`or`|하나라도 참이면 참|
|`not`|참/거짓 반전|

예시:

```python
age = 25
salary = 3500

if age >= 25 and salary >= 3500:
    print("조건 만족")
```

## 11. `and`

두 조건이 모두 참이어야 한다.

```python
price = 30000
card = "python"

if price > 20000 and card == "python":
    print("배송료가 없습니다.")
else:
    print("배송료는 3000원입니다.")
```

## 12. `or`

둘 중 하나만 참이어도 된다.

```python
country = "USA"
province = "California"

if country != "Korea" or province == "Jeju":
    shipping_cost = 8000
```

## 13. `not`

조건 결과를 반대로 바꾼다.

```python
is_member = False

if not is_member:
    print("회원이 아닙니다.")
```

## 14. `elif`

조건이 여러 개일 때 사용한다.

```python
score = int(input("성적을 입력하시오: "))

if score >= 90:
    print("학점 A")
elif score >= 80:
    print("학점 B")
elif score >= 70:
    print("학점 C")
elif score >= 60:
    print("학점 D")
else:
    print("학점 F")
```

`elif`는 위 조건이 거짓일 때 다음 조건을 검사한다.

## 15. 중첩 조건문

조건문 안에 조건문을 넣을 수 있다.

```python
country = input("배송지: ")
price = int(input("상품의 가격: "))

if country == "korea":
    if price >= 20000:
        shipping_cost = 0
    else:
        shipping_cost = 3000
else:
    if price >= 100000:
        shipping_cost = 0
    else:
        shipping_cost = 8000

print("배송비 =", shipping_cost)
```

조건이 단계적으로 나뉘는 경우에 사용한다.

## 16. 조건부 표현식

간단한 조건문은 한 줄로 쓸 수 있다.

```python
shipping_cost = 0 if price >= 20000 else 3000
```

예시:

```python
x = 10
y = 20

max_value = x if x > y else y
```

## 대표 예제: 세일 가격 계산

```python
price = int(input("정가를 입력하시오: "))

if price >= 100:
    dis_rate = 0.85
    print("10층에서 사은품을 받아가세요.")
else:
    dis_rate = 0.90

dis_price = dis_rate * price

print("할인된 상품의 가격 =", dis_price)
```

## 대표 예제: 물의 상태 출력

```python
temp = float(input("온도를 입력하시오: "))

if temp <= 0:
    print("물의 상태는 얼음입니다.")
elif temp > 0 and temp < 100:
    print("물의 상태는 액체입니다.")
else:
    print("물의 상태는 기체입니다.")
```

## 대표 예제: 리히터 규모 판별

```python
scale = float(input("리히터 규모를 입력하시오: "))

if scale >= 8.0:
    print("대부분의 구조물이 파괴됩니다.")
elif scale >= 7.0:
    print("지표면에 균열이 발생합니다.")
elif scale >= 4.0:
    print("빈약한 건물에 큰 피해가 있습니다.")
elif scale >= 2.0:
    print("물건들이 흔들리거나 떨어집니다.")
else:
    print("지진계에 의해서만 탐지 가능합니다.")
```

## 오늘의 핵심 정리

```text
조건문은 프로그램의 흐름을 갈라놓는 문법이다.
if는 조건이 참일 때 실행된다.
else는 조건이 거짓일 때 실행된다.
elif는 여러 조건을 순서대로 검사할 때 사용한다.
비교 연산자는 두 값을 비교한다.
논리 연산자는 여러 조건을 결합한다.
들여쓰기가 블록을 결정한다.
```

## 헷갈린 점 정리

### `=`와 `==`

```python
x = 10      # 대입
x == 10     # 비교
```

### `if`와 `elif`

여러 조건을 독립적으로 검사하려면 `if`를 여러 번 쓴다.

```python
if x > 0:
    print("양수")

if x % 2 == 0:
    print("짝수")
```

하나의 분기에서 하나만 선택하려면 `if-elif-else`를 쓴다.

```python
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C 이하")
```

### `and`와 `or`

```text
and: 둘 다 참이어야 함
or: 하나만 참이어도 됨
```

## 내 식으로 이해

조건문은 프로그램 안에 갈림길을 만드는 것이다.

사람이 상황에 따라 판단하듯이, 프로그램도 조건문을 통해 입력값이나 상황에 따라 다른 동작을 수행한다.

PLC에서 센서 조건에 따라 모터를 켜거나 끄는 것처럼, Python에서는 `if` 조건에 따라 특정 코드를 실행하거나 건너뛴다.
