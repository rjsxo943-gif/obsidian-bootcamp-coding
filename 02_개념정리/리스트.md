## 한 줄 정의

리스트는 여러 값을 순서대로 저장하는 자료구조이다.

여러 개의 데이터를 하나의 변수에 묶어서 관리할 수 있다.

```python
numbers = [10, 20, 30, 40]
```

## 핵심 개념

- 리스트 생성
    
- 인덱스
    
- 음수 인덱스
    
- 요소 변경
    
- 리스트 순회
    
- `append()`
    
- `insert()`
    
- `remove()`
    
- `pop()`
    
- `sort()`
    
- `sorted()`
    
- 슬라이싱
    
- 얕은 복사
    
- 깊은 복사
    
- 리스트 함축
    
- 2차원 리스트
    

## 1. 리스트 만들기

```python
numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "grape"]
mixed = [1, "hello", 3.14]
```

Python 리스트는 다양한 자료형을 함께 저장할 수 있다.  
다만 보통은 같은 성격의 데이터끼리 묶는 것이 좋다.

## 2. 빈 리스트

```python
my_list = []
```

빈 리스트를 만든 뒤 값을 추가할 수 있다.

```python
my_list.append(10)
my_list.append(20)

print(my_list)
```

## 3. 인덱스

리스트의 각 요소는 위치 번호로 접근한다.  
Python의 인덱스는 0부터 시작한다.

```python
fruits = ["apple", "banana", "grape"]

print(fruits[0])
print(fruits[1])
print(fruits[2])
```

결과:

```text
apple
banana
grape
```

## 4. 음수 인덱스

음수 인덱스는 뒤에서부터 접근한다.

```python
fruits = ["apple", "banana", "grape"]

print(fruits[-1])
print(fruits[-2])
```

|인덱스|의미|
|---|---|
|`-1`|마지막 요소|
|`-2`|뒤에서 두 번째 요소|

## 5. 리스트 요소 변경

리스트는 값을 변경할 수 있다.

```python
subjects = ["영어", "수학", "사회", "과학"]

subjects[-1] = "컴퓨터"

print(subjects)
```

## 6. 리스트 순회

리스트의 모든 값을 하나씩 꺼낼 수 있다.

```python
temps = [28, 31, 33, 35, 27]

for temp in temps:
    print(temp)
```

인덱스가 필요하면 `range(len())`를 사용할 수 있다.

```python
for i in range(len(temps)):
    print(i, temps[i])
```

## 7. `append()`

리스트의 맨 뒤에 값을 추가한다.

```python
heroes = []

heroes.append("아이언맨")
heroes.append("토르")

print(heroes)
```

## 8. `insert()`

원하는 위치에 값을 삽입한다.

```python
heroes = ["아이언맨", "토르", "헐크"]

heroes.insert(1, "스파이더맨")

print(heroes)
```

## 9. `remove()`

값을 기준으로 삭제한다.

```python
heroes = ["아이언맨", "토르", "헐크"]

heroes.remove("토르")

print(heroes)
```

삭제할 값이 없으면 오류가 난다.  
그래서 먼저 확인하는 것이 안전하다.

```python
if "토르" in heroes:
    heroes.remove("토르")
```

## 10. `pop()`

인덱스를 기준으로 값을 꺼내고 삭제한다.

```python
heroes = ["아이언맨", "토르", "헐크"]

hero = heroes.pop(1)

print(hero)
print(heroes)
```

인덱스를 생략하면 마지막 요소를 꺼낸다.

```python
last = heroes.pop()
```

## 11. 리스트 탐색

값이 리스트 안에 있는지 확인할 수 있다.

```python
heroes = ["아이언맨", "토르", "헐크"]

if "헐크" in heroes:
    print("있음")
```

위치를 찾으려면 `index()`를 사용한다.

```python
print(heroes.index("헐크"))
```

## 12. 리스트의 최댓값, 최솟값, 합계

```python
values = [10, 20, 30, 40, 50]

print(sum(values))
print(max(values))
print(min(values))
print(len(values))
```

|함수|의미|
|---|---|
|`sum()`|합계|
|`max()`|최댓값|
|`min()`|최솟값|
|`len()`|길이|

평균:

```python
avg = sum(values) / len(values)
```

## 13. 정렬

리스트 자체를 정렬하려면 `sort()`를 사용한다.

```python
numbers = [3, 2, 1, 5, 4]

numbers.sort()

print(numbers)
```

역순 정렬:

```python
numbers.sort(reverse=True)
```

원본을 유지하고 정렬 결과만 새로 얻으려면 `sorted()`를 사용한다.

```python
numbers = [3, 2, 1, 5, 4]

new_numbers = sorted(numbers)

print(numbers)
print(new_numbers)
```

## 14. 리스트 합치기와 반복

```python
heroes1 = ["아이언맨", "토르"]
heroes2 = ["헐크", "스칼렛 위치"]

avengers = heroes1 + heroes2

print(avengers)
```

리스트 반복:

```python
numbers = [1, 2, 3] * 3
print(numbers)
```

## 15. 리스트 복사 주의

```python
temps = [28, 31, 33, 35]
values = temps

values[0] = 99

print(temps)
```

`values = temps`는 리스트를 복사한 것이 아니라 같은 리스트를 함께 가리키는 것이다.

그래서 하나를 바꾸면 다른 쪽도 바뀐다.

## 16. 리스트 복사 방법

새 리스트를 만들려면 다음처럼 한다.

```python
values = list(temps)
```

또는 슬라이싱을 사용할 수 있다.

```python
values = temps[:]
```

## 17. 슬라이싱

리스트의 일부를 잘라낼 수 있다.

```python
numbers = [10, 20, 30, 40, 50, 60]

print(numbers[0:3])
print(numbers[:3])
print(numbers[3:])
print(numbers[:])
```

역순:

```python
print(numbers[::-1])
```

하나씩 건너뛰기:

```python
print(numbers[::2])
```

## 18. 리스트 일부 변경

```python
lst = [1, 2, 3, 4, 5]

lst[0:3] = ["white", "blue", "red"]

print(lst)
```

리스트 전체 삭제:

```python
lst[:] = []
```

## 19. 리스트가 함수로 전달될 때

리스트는 가변 객체이므로 함수 안에서 변경하면 원본도 바뀔 수 있다.

```python
def modify(values):
    values[0] = 99

numbers = [1, 2, 3]

modify(numbers)

print(numbers)
```

결과:

```text
[99, 2, 3]
```

## 20. 리스트 함축

리스트를 간단하게 만들 수 있는 문법이다.

```python
squares = [x * x for x in range(10)]

print(squares)
```

조건을 붙일 수도 있다.

```python
numbers = [x for x in range(100) if x % 2 == 0 and x % 3 == 0]

print(numbers)
```

## 21. 2차원 리스트

2차원 리스트는 리스트 안에 리스트가 들어간 구조이다.

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
```

요소 접근:

```python
print(matrix[0][0])
print(matrix[1][2])
```

## 22. 2차원 리스트 순회

```python
for r in range(len(matrix)):
    for c in range(len(matrix[0])):
        print(matrix[r][c], end=" ")
    print()
```

## 대표 예제: 성적 처리 프로그램

```python
STUDENTS = 5
scores = []
count = 0

for i in range(STUDENTS):
    value = int(input("성적을 입력하시오: "))
    scores.append(value)

print("성적 평균 =", sum(scores) / len(scores))
print("최대점수 =", max(scores))
print("최소점수 =", min(scores))

for score in scores:
    if score >= 80:
        count += 1

print("80점 이상 =", count)
```

## 대표 예제: 스택 흉내내기

```python
stack = []

for i in range(3):
    fruit = input("과일을 입력하시오: ")
    stack.append(fruit)

for i in range(3):
    print(stack.pop())
```

스택은 나중에 들어온 값이 먼저 나가는 구조이다.

```text
LIFO: Last In First Out
```

## 대표 예제: 친구 관리 프로그램

```python
menu = 0
friends = []

while menu != 9:
    print("--------------------")
    print("1. 친구 리스트 출력")
    print("2. 친구추가")
    print("3. 친구삭제")
    print("4. 이름변경")
    print("9. 종료")

    menu = int(input("메뉴를 선택하시오: "))

    if menu == 1:
        print(friends)
    elif menu == 2:
        name = input("이름을 입력하시오: ")
        friends.append(name)
    elif menu == 3:
        del_name = input("삭제하고 싶은 이름을 입력하시오: ")

        if del_name in friends:
            friends.remove(del_name)
        else:
            print("이름이 발견되지 않았음")
    elif menu == 4:
        old_name = input("변경하고 싶은 이름을 입력하시오: ")

        if old_name in friends:
            index = friends.index(old_name)
            new_name = input("새로운 이름을 입력하시오: ")
            friends[index] = new_name
        else:
            print("이름이 발견되지 않았음")
```

## 오늘의 핵심 정리

```text
리스트는 여러 값을 순서대로 저장하는 자료구조이다.
인덱스는 0부터 시작한다.
-1은 마지막 요소를 의미한다.
append()는 값을 추가한다.
remove()는 값을 기준으로 삭제한다.
pop()은 위치를 기준으로 꺼내고 삭제한다.
sort()는 원본 리스트를 정렬한다.
sorted()는 정렬된 새 리스트를 반환한다.
슬라이싱은 리스트 일부를 잘라낸다.
리스트는 가변 객체이므로 함수 안에서 변경될 수 있다.
```

## 헷갈린 점 정리

### `append()`와 `insert()`

|메서드|의미|
|---|---|
|`append()`|맨 뒤에 추가|
|`insert()`|원하는 위치에 추가|

### `remove()`와 `pop()`

|메서드|기준|
|---|---|
|`remove(value)`|값 기준 삭제|
|`pop(index)`|인덱스 기준 삭제|

### 얕은 복사

```python
b = a
```

같은 리스트를 가리킨다.

### 복사본 만들기

```python
b = a[:]
b = list(a)
```

새 리스트를 만든다.

## 내 식으로 이해

리스트는 여러 데이터를 한 줄로 묶어서 관리하는 도구이다.

전자공학에서 여러 센서값을 배열로 저장하거나, PLC에서 여러 입력 상태를 순서대로 관리하는 것처럼, Python에서는 리스트를 사용해 여러 값을 한 번에 다룬다.

반복문과 리스트를 함께 쓰면 많은 데이터를 자동으로 처리할 수 있다.
