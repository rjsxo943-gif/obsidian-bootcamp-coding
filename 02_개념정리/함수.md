## 한 줄 정의

함수는 반복해서 사용할 코드를 이름 붙여 묶어둔 것이다.

특정 작업을 수행하는 코드 덩어리에 이름을 붙이고, 필요할 때 호출해서 사용할 수 있다.

```python
def get_area(radius):
    area = 3.14 * radius ** 2
    return area
```

## 1. 함수가 필요한 이유

프로그램을 작성하다 보면 같은 코드를 여러 번 써야 하는 경우가 있다.

이때 함수를 사용하면 같은 코드를 반복해서 작성하지 않고, 필요할 때 호출해서 사용할 수 있다.

함수의 장점:

```text
코드 중복을 줄인다.
코드를 기능별로 나눌 수 있다.
한 번 만든 함수를 다른 곳에서도 재사용할 수 있다.
복잡한 문제를 작은 단위로 나눌 수 있다.
```

## 2. 함수 정의

함수는 `def` 키워드로 정의한다.

```python
def 함수이름(매개변수):
    실행문
    return 반환값
```

예시:

```python
def get_area(radius):
    area = 3.14 * radius ** 2
    return area
```

## 3. 함수 호출

함수를 사용하려면 함수 이름을 적고 괄호를 붙인다.

```python
result = get_area(3)
print(result)
```

함수가 호출되면 함수 안의 코드가 실행되고, 실행이 끝나면 호출한 위치로 돌아온다.

## 4. 인수와 매개변수

|용어|의미|
|---|---|
|인수|함수를 호출할 때 넣는 값|
|매개변수|함수가 값을 받는 변수|

```python
def get_area(radius):
    return 3.14 * radius ** 2

result = get_area(3)
```

여기서 `3`은 인수이고, `radius`는 매개변수이다.

## 5. 반환값과 `return`

`return`은 함수의 결과를 함수 밖으로 돌려준다.

```python
def add(a, b):
    return a + b

result = add(10, 20)
print(result)
```

`return`이 없으면 함수는 결과를 밖으로 돌려주지 않는다.

## 6. `print()`와 `return` 차이

| 구분        | 의미           |
| --------- | ------------ |
| `print()` | 화면에 출력       |
| `return`  | 함수 밖으로 값을 반환 |

```python
def add_print(a, b):
    print(a + b)

def add_return(a, b):
    return a + b
```

`print()`는 보여주는 것이고, `return`은 값을 다시 사용할 수 있게 돌려주는 것이다.

## 7. 여러 개의 값 반환

Python 함수는 여러 값을 반환할 수 있다.

```python
def get_input():
    return 2, 3

x, y = get_input()

print(x)
print(y)
```

실제로는 튜플 형태로 반환된다.

## 8. 함수의 순서

Python은 위에서 아래로 코드를 실행한다.

따라서 함수를 호출하기 전에 함수가 먼저 정의되어 있어야 한다.

```python
def get_area(radius):
    return 3.14 * radius ** 2

result = get_area(3)
print(result)
```

다만 함수 안에서 다른 함수를 호출하는 경우에는, 실제 실행 시점 전에만 정의되어 있으면 된다.

```python
def main():
    result = get_area(3)
    print(result)

def get_area(radius):
    return 3.14 * radius ** 2

main()
```

## 9. 기본값 인수

매개변수에 기본값을 줄 수 있다.

```python
def greet(name, msg="별일없죠?"):
    print("안녕", name + ", " + msg)

greet("영희")
greet("철수", "반가워")
```

값을 넣지 않으면 기본값이 사용된다.

## 10. 키워드 인수

인수 이름을 직접 지정해서 값을 전달할 수 있다.

```python
def sub(x, y, z):
    print("x =", x, "y =", y, "z =", z)

sub(x=10, y=20, z=30)
```

일부만 키워드 인수로 사용할 수도 있다.

```python
sub(10, y=20, z=30)
```

주의할 점:

```python
sub(x=10, 20, 30)
```

이런 형태는 오류이다.  
키워드 인수 뒤에 위치 인수가 올 수 없다.

## 11. 가변 인수 `*args`

인수의 개수가 정해져 있지 않을 때 `*args`를 사용한다.

```python
def add(*numbers):
    total = 0

    for n in numbers:
        total += n

    return total

print(add(10, 20))
print(add(10, 20, 30))
```

`*args`는 여러 값을 튜플처럼 받는다.

## 12. 가변 키워드 인수 `**kwargs`

키워드 인수를 여러 개 받을 때 사용한다.

```python
def myfunc(**kwargs):
    result = ""

    for value in kwargs.values():
        result += value

    return result

print(myfunc(a="Hi!", b="Mr.", c="Kim"))
```

`**kwargs`는 딕셔너리 형태로 전달된다.

## 13. 언패킹

리스트나 튜플을 함수 인수로 풀어서 전달할 수 있다.

```python
def add(a, b, c):
    return a + b + c

numbers = [1, 2, 3]

print(add(*numbers))
```

딕셔너리는 `**`로 언패킹할 수 있다.

## 14. 지역 변수

함수 안에서 만든 변수는 함수 안에서만 사용할 수 있다.

```python
def myfunc():
    x = 100
    print(x)

myfunc()
```

여기서 `x`는 지역 변수이다.

## 15. 전역 변수

함수 밖에서 만든 변수는 전역 변수이다.

```python
gx = 100

def myfunc():
    print(gx)

myfunc()
print(gx)
```

함수 안에서 전역 변수의 값을 읽을 수 있다.

## 16. 함수 안에서 전역 변수 변경

함수 안에서 전역 변수와 같은 이름에 값을 대입하면 새로운 지역 변수가 만들어진다.

```python
gx = 100

def myfunc():
    gx = 200
    print(gx)

myfunc()
print(gx)
```

결과:

```text
200
100
```

전역 변수를 직접 변경하려면 `global`을 사용한다.

```python
gx = 100

def myfunc():
    global gx
    gx = 200

myfunc()
print(gx)
```

하지만 전역 변수를 자주 변경하는 코드는 복잡해질 수 있으므로 주의해야 한다.

## 17. 재귀 호출

재귀 호출은 함수가 자기 자신을 호출하는 것이다.

```python
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))
```

재귀 함수에는 반드시 종료 조건이 필요하다.

## 대표 예제: 피자 크기 비교

```python
def get_area(radius):
    if radius > 0:
        area = 3.14 * radius ** 2
    else:
        area = 0

    return area

def main():
    print("20cm 피자 2개의 면적:", get_area(20) + get_area(20))
    print("30cm 피자 1개의 면적:", get_area(30))

main()
```

## 대표 예제: 주급 계산

```python
def weeklyPay(rate, hour):
    money = 0

    if hour > 30:
        money = rate * 30 + 1.5 * rate * (hour - 30)
    else:
        money = rate * hour

    return money

rate = int(input("시급을 입력하시오: "))
hour = int(input("근무 시간을 입력하시오: "))

print("주급은", weeklyPay(rate, hour))
```

## 대표 예제: 온도 변환 프로그램

```python
def menu():
    print("1. 섭씨 온도 -> 화씨 온도")
    print("2. 화씨 온도 -> 섭씨 온도")
    print("3. 종료")

    selection = int(input("메뉴를 선택하세요: "))
    return selection

def ctof(c):
    return c * 9.0 / 5.0 + 32

def ftoc(f):
    return (f - 32.0) * 5.0 / 9.0

def input_c():
    return float(input("섭씨 온도를 입력하시오: "))

def input_f():
    return float(input("화씨 온도를 입력하시오: "))

def main():
    while True:
        index = menu()

        if index == 1:
            t = input_c()
            print("화씨 온도 =", ctof(t))
        elif index == 2:
            t = input_f()
            print("섭씨 온도 =", ftoc(t))
        else:
            break

main()
```

## 오늘의 핵심 정리

```text
함수는 코드를 이름 붙여 묶은 것이다.
def로 함수를 정의한다.
함수 이름과 괄호로 함수를 호출한다.
인수는 호출할 때 넣는 값이다.
매개변수는 함수가 값을 받는 변수이다.
return은 함수 결과를 밖으로 돌려준다.
지역 변수는 함수 안에서만 사용할 수 있다.
전역 변수는 함수 밖에서 만들어진 변수이다.
```
