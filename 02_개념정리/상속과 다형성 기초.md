# 상속과 다형성 기초

## 핵심 요약

상속은 기존 클래스의 속성과 기능을 물려받아 새로운 클래스를 만드는 것이다.  
다형성은 같은 메소드 이름을 호출해도 객체 종류에 따라 다르게 동작하는 것이다.

---

## 상속 예시

```python
class Car:
    def __init__(self, make, model, color, speed):
        self.make = make
        self.model = model
        self.color = color
        self.speed = speed
```

```python
class ElectricCar(Car):
    def __init__(self, make, model, color, price, battery_capacity):
        super().__init__(make, model, color, price)
        self.battery_capacity = battery_capacity
```

`ElectricCar`는 `Car`를 상속받는다.

```text
전기차는 자동차이다.
그러므로 자동차의 기본 속성을 물려받고,
전기차만의 배터리 용량을 추가한다.
```

---

## super()

```python
super().__init__(make, model, color, price)
```

부모 클래스의 생성자를 호출한다.  
즉 부모가 가지고 있는 초기화 코드를 재사용한다.

## 다형성 예시

```python
class Shape:
    def area(self):
        raise NotImplementedError
```

```python
class Circle(Shape):
    def area(self):
        return math.pi * (self.radius ** 2)
```

```python
class Rectangle(Shape):
    def area(self):
        return self.width * self.height
```

같은 `area()`지만 원과 사각형의 넓이 계산 방식은 다르다.

```python
for shape in shapeList:
    print(shape.area())
```

`shape`가 원이면 원의 넓이를 계산하고, 사각형이면 사각형의 넓이를 계산한다.

---

## 내 식으로 이해

상속은 “기본형을 가져와서 확장하는 것”이다.  
다형성은 “같은 명령을 내려도 대상에 따라 다르게 반응하는 것”이다.

예를 들어 `area()`라는 말은 같지만, 원에게 말하면 원 넓이를 계산하고 사각형에게 말하면 사각형 넓이를 계산한다.
