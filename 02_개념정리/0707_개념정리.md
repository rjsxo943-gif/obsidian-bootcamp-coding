## 1. 외부 라이브러리

외부 라이브러리는 Python 기본 기능에 없는 기능을 추가로 사용하기 위해 설치하는 코드 묶음이다.

예를 들어 음성 변환, 데이터 분석, 머신러닝, 그래프 시각화는 전부 외부 라이브러리를 사용하면 쉽게 구현할 수 있다.

```text
gTTS → 텍스트를 음성으로 변환
pandas → 표 형태 데이터 처리
numpy → 수치 계산
scikit-learn → 머신러닝
plotly → 시각화
```

관련 코드: [[0707_코드해설#2. pandas와 numpy 불러오기]]

## 2. pip install

`pip install`은 Python 패키지를 설치하는 명령어이다.

```powershell
pip install gtts
pip install pandas
pip install scikit-learn
pip install plotly
```

중요한 점은 설치하는 Python 환경과 실행하는 Python 환경이 같아야 한다는 것이다.

> [!tip]
> 설치는 터미널에서 하지만, 실제 사용은 코드 안의 `import`에서 시작된다.

## 3. 가상환경

가상환경은 프로젝트마다 Python 패키지를 따로 관리하기 위한 공간이다.

```text

협업 시 같은 환경 재현
```

이 경우 패키지가 어디에 설치되는지 헷갈릴 수 있으므로 하나의 환경 기준으로 정리하는 것이 좋다.

## 4. TTS

TTS는 Text To Speech의 약자이다.

텍스트를 사람 목소리처럼 음성으로 변환하는 기술이다.

```python
from gtts import gTTS

tts = gTTS(text="안녕하세요", lang="ko")
tts.save("hello.mp3")
```

| 개념 | 의미 |
|---|---|
| TTS | 텍스트를 음성으로 변환 |
| gTTS | Google Text-to-Speech 기반 Python 라이브러리 |
| mp3 | 음성 파일 형식 |
| save() | 변환 결과를 파일로 저장 |

관련 코드: [[0707_코드해설#1. gTTS로 음성 파일 만들기]]

## 10. describe()

`describe()`는 숫자형 데이터의 기본 통계를 요약해준다.

```python
titanic.describe()
```

확인할 수 있는 값:

```text
count → 데이터 개수
mean → 평균
std → 표준편차
min → 최솟값
25% → 1사분위수
50% → 중앙값
75% → 3사분위수
max → 최댓값
```

관련 코드: [[0707_코드해설#7. 데이터 통계 요약]]

## 11. iloc

`iloc`는 정수 위치를 기준으로 행과 열을 선택한다.

```python
titanic.iloc[20:23, 5:7]
```

해석:

```text
20:23 → 20번, 21번, 22번 행
5:7 → 5번, 6번 열
```

Python 슬라이싱처럼 끝 번호는 포함되지 않는다.

관련 코드: [[0707_코드해설#8. iloc로 행과 열 선택]]

## 12. pivot_table

`pivot_table()`은 데이터를 기준별로 요약할 때 사용한다.

```python
table = pd.pivot_table(
    titanic,
    index="Sex",
    columns="Pclass",
    values="Age",
    aggfunc=np.mean
)
```

| 인자 | 의미 |
|---|---|
| `index` | 행 기준 |
| `columns` | 열 기준 |
| `values` | 계산할 값 |
| `aggfunc` | 계산 방식 |

이 코드는 **성별과 객실 등급별 평균 나이**를 계산한다.

## 15. 머신러닝 기본 구조

머신러닝은 데이터를 이용해 모델이 규칙을 학습하고, 새로운 데이터에 대해 결과를 예측하는 방식이다.

```text
입력 데이터 X
정답 데이터 y
모델 생성
학습 fit()
예측 predict()
```

## 16. SVM

SVM은 Support Vector Machine의 약자이다.

분류 문제에서 자주 사용되는 머신러닝 알고리즘이다.

```python
s = svm.SVC(gamma=0.1, C=10)
```

| 파라미터 | 의미 |
|---|---|
| `C` | 오분류를 얼마나 허용할지 조절 |
| `gamma` | 하나의 데이터가 영향을 주는 범위 |

현재 단계에서는 `C`와 `gamma`를 완벽히 이해하기보다, 모델 성능을 조절하는 설정값이라고 이해하면 된다.

## 17. fit

`fit()`은 모델을 학습시키는 함수이다.

```python
s.fit(iris.data, iris.target)
```

해석:

```text
iris.data → 입력 데이터
iris.target → 정답 데이터
```

모델은 입력과 정답의 관계를 학습한다.

---

## 18. predict

`predict()`는 학습된 모델로 새 데이터를 예측하는 함수이다.

```python
res = s.predict(new_d)
```

결과는 정답 라벨 번호로 나온다.

```python
iris.target_names[res]
```

이렇게 하면 번호를 실제 품종 이름으로 바꿔 볼 수 있다.

## 20. 라이브러리 역할 구분

오늘 가장 중요한 개념 중 하나는 라이브러리 역할을 구분하는 것이다.

| 라이브러리 | 역할 |
|---|---|
| `gTTS` | 텍스트를 음성으로 변환 |
| `pandas` | 표 형태 데이터 처리 |
| `numpy` | 수치 계산 |
| `sklearn.datasets` | 예제 데이터셋 로드 |
| `sklearn.svm` | 머신러닝 모델 생성 |
| `plotly.express` | 그래프 시각화 |

> [!important]
> `pandas`는 데이터 처리용이고, `plotly`는 시각화용이다.  
> 따라서 `pd.scatter_3d()`가 아니라 `px.scatter_3d()`를 사용한다.
