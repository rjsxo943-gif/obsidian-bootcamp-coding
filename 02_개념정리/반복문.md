## 한 줄 정의

반복문은 같은 동작을 여러 번 실행하는 문법이다.

반복적인 작업을 사람이 직접 여러 번 작성하지 않고, 컴퓨터가 자동으로 수행하게 만든다.

```python
for i in range(5):
    print("Hello")
```

## 핵심 개념

- 반복 구조
    
- `for`
    
- `while`
    
- `range()`
    
- 리스트 반복
    
- 중첩 반복문
    
- 무한 루프
    
- `break`
    
- `continue`
    

## 1. 반복문이 필요한 이유

같은 문장을 여러 번 출력해야 한다고 하자.

```python
print("방문을 환영합니다!")
print("방문을 환영합니다!")
print("방문을 환영합니다!")
print("방문을 환영합니다!")
print("방문을 환영합니다!")
```

반복문을 쓰면 이렇게 줄일 수 있다.

```python
for i in range(5):
    print("방문을 환영합니다!")
```

반복문은 반복적이고 단순한 작업을 자동화할 때 사용한다.

## 2. 반복문의 종류

|종류|설명|
|---|---|
|`for`문|정해진 횟수만큼 반복|
|`while`문|조건이 참인 동안 반복|

## 3. `for`문

`for`문은 리스트나 범위에서 값을 하나씩 꺼내며 반복한다.

```python
for i in [1, 2, 3, 4, 5]:
    print(i)
```

## 4. `range()`

`range()`는 일정한 범위의 정수를 만들어준다.

```python
for i in range(5):
    print(i)
```

결과:

```text
0
1
2
3
4
```

`range(5)`는 0부터 4까지이다.  
끝값 5는 포함되지 않는다.

## 5. `range(start, end)`

```python
for i in range(1, 6):
    print(i)
```

결과:

```text
1
2
3
4
5
```

`range(1, 6)`은 1부터 5까지이다.

## 6. `range(start, end, step)`

```python
for i in range(1, 10, 2):
    print(i)
```

결과:

```text
1
3
5
7
9
```

세 번째 값은 증가폭이다.

역순도 가능하다.

```python
for i in range(10, 0, -1):
    print(i)
```

## 7. 1부터 n까지의 합

```python
total = 0
n = 10

for i in range(1, n + 1):
    total += i

print("합 =", total)
```

`total += i`는 `total = total + i`와 같다.

## 8. 구구단 출력

```python
dan = int(input("원하는 단은: "))

for i in range(1, 10):
    print(f"{dan} * {i} = {dan * i}")
```

## 9. `while`문

`while`문은 조건이 참인 동안 계속 반복한다.

```python
i = 1
total = 0

while i <= 10:
    total += i
    i += 1

print("합계는", total)
```

`while`은 조건이 계속 참이면 무한히 반복할 수 있으므로, 조건을 바꾸는 코드가 필요하다.

## 10. `for`와 `while` 차이

|구분|사용 상황|
|---|---|
|`for`|반복 횟수가 정해져 있을 때|
|`while`|특정 조건이 만족되는 동안 반복할 때|

예를 들어 10번 반복은 `for`가 편하다.

```python
for i in range(10):
    print(i)
```

비밀번호가 맞을 때까지 반복은 `while`이 편하다.

```python
password = ""

while password != "pythonisfun":
    password = input("암호를 입력하시오: ")

print("로그인 성공")
```

## 11. 무한 루프

조건이 계속 참이면 반복문이 끝나지 않는다.

```python
while True:
    command = input("명령 입력: ")

    if command == "q":
        break
```

`while True`는 의도적으로 무한 반복할 때 사용한다.  
종료 조건으로 `break`를 함께 쓰는 경우가 많다.

## 12. `break`

`break`는 반복문을 즉시 종료한다.

```python
while True:
    light = input("신호등 색상을 입력하시오: ")

    if light == "green":
        break

print("전진!!")
```

## 13. `continue`

`continue`는 이번 반복의 나머지 코드를 건너뛰고 다음 반복으로 넘어간다.

```python
for i in range(1, 11):
    if i % 3 == 0:
        continue

    print(i, end=" ")
```

3의 배수는 출력하지 않고 넘어간다.

## 14. 중첩 반복문

반복문 안에 반복문이 들어간 구조이다.

```python
for y in range(5):
    for x in range(10):
        print("*", end="")
    print()
```

결과:

```text
**********
**********
**********
**********
**********
```

## 15. 별 찍기

```python
for y in range(1, 6):
    for x in range(y):
        print("*", end="")
    print()
```

결과:

```text
*
**
***
****
*****
```

별 찍기는 반복문의 구조를 눈으로 확인하기 좋은 연습이다.

## 16. 모든 조합 구하기

```python
adj = ["small", "medium", "large"]
nouns = ["apple", "banana", "grape"]

for x in adj:
    for y in nouns:
        print(x, y)
```

중첩 반복문은 모든 조합을 만들 때 자주 사용된다.

## 대표 예제: 숫자 맞추기 게임

```python
import random

tries = 0
guess = 0
answer = random.randint(1, 100)

print("1부터 100 사이의 숫자를 맞추시오")

while guess != answer:
    guess = int(input("숫자를 입력하시오: "))
    tries += 1

    if guess < answer:
        print("너무 낮음!")
    elif guess > answer:
        print("너무 높음!")

print("축하합니다. 시도횟수 =", tries)
```

## 대표 예제: 산수 문제 발생기

```python
import random

flag = True

while flag:
    x = random.randint(1, 100)
    y = random.randint(1, 100)

    answer = int(input(f"{x} + {y} = "))

    if answer == x + y:
        print("잘했어요!!")
    else:
        print("틀렸어요. 하지만 다음번에는 잘할 수 있죠?")
        flag = False
```

## 대표 예제: 소수 찾기

```python
N_PRIMES = 50
number = 2
count = 0

while count < N_PRIMES:
    divisor = 2
    prime = True

    while divisor < number:
        if number % divisor == 0:
            prime = False
            break
        divisor += 1

    if prime:
        count += 1
        print(number, end=" ")

    number += 1
```

## 오늘의 핵심 정리

```text
반복문은 같은 코드를 여러 번 실행하는 문법이다.
for문은 반복 횟수가 정해져 있을 때 좋다.
while문은 조건이 참인 동안 반복할 때 좋다.
range()는 반복에 사용할 정수 범위를 만든다.
break는 반복문을 종료한다.
continue는 다음 반복으로 넘어간다.
중첩 반복문은 반복문 안에 반복문이 들어간 구조이다.
```

## 헷갈린 점 정리

### `range(5)`

```text
0, 1, 2, 3, 4
```

5번 반복하지만 5는 포함하지 않는다.

### `range(1, 6)`

```text
1, 2, 3, 4, 5
```

끝값 6은 포함하지 않는다.

### `break`와 `continue`

|키워드|의미|
|---|---|
|`break`|반복문 종료|
|`continue`|이번 반복만 건너뛰기|

## 내 식으로 이해

반복문은 단순 반복 작업을 컴퓨터에게 맡기는 것이다.

전자공학이나 PLC에서 같은 동작을 일정 횟수 반복하거나, 조건이 만족될 때까지 동작을 유지하는 것과 비슷하다.

Python에서는 `for`와 `while`을 사용해서 이런 반복 흐름을 만든다.

-  1부터 100까지 합을 구할 수 있다.
    
-  구구단을 출력할 수 있다.
