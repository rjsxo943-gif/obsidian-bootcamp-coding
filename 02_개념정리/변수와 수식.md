## 한 줄 정의

변수는 값을 저장하기 위해 이름을 붙인 공간이고, 수식은 값을 계산하기 위한 표현식이다.

Python에서는 변수에 값을 대입하면 자동으로 변수가 생성된다.

```python
x = 100
y = 200
total = x + y

print(total)
```

## 1. 변수

변수는 값을 저장하기 위한 이름이다.

```python
x = 100
print(x)
```

여기서 `x`는 변수 이름이고, `100`은 변수에 저장된 값이다.

Python에서는 변수의 자료형을 미리 선언하지 않는다.  
값을 넣는 순간 Python이 알아서 자료형을 판단한다.

```python
radius = 10
radius = 10.5
radius = "Unknown"
```

다만 하나의 변수에 계속 다른 종류의 값을 넣으면 코드가 헷갈릴 수 있으므로 좋은 습관은 아니다.

## 2. 자료형

자료형은 데이터의 종류이다.

대표적인 자료형은 다음과 같다.

|자료형|의미|예시|
|---|---|---|
|`int`|정수|`10`|
|`float`|실수|`3.14`|
|`str`|문자열|`"hello"`|
|`bool`|참/거짓|`True`, `False`|
|`list`|여러 값을 저장|`[1, 2, 3]`|

자료형을 확인하려면 `type()`을 사용한다.

```python
print(type(10))
print(type(3.14))
print(type("hello"))
```

## 3. Python 변수의 특징

Python에서 변수는 실제 값을 직접 담는 상자라기보다, 객체를 가리키는 이름에 가깝다.

```python
x = 100
```

이 코드는 `100`이라는 객체를 만들고, `x`가 그 객체를 가리키게 하는 것이다.

## 4. 주석

주석은 코드 실행에는 영향을 주지 않고 설명을 적는 부분이다.

```python
# 반지름을 저장한다.
radius = 10

# 원의 넓이를 계산한다.
area = 3.14 * radius * radius
```

주석은 나중에 코드를 다시 볼 때 이해하기 쉽게 해준다.

## 5. 상수

Python에는 엄격한 상수 문법은 없지만, 보통 변하지 않는 값은 대문자로 작성한다.

```python
PI = 3.14
TAX_RATE = 0.1
```

대문자로 썼다고 실제로 변경이 막히는 것은 아니다.  
다만 “이 값은 바꾸지 말자”라는 약속이다.

## 6. 산술 연산자

|연산자|의미|예시|
|---|---|---|
|`+`|덧셈|`10 + 3`|
|`-`|뺄셈|`10 - 3`|
|`*`|곱셈|`10 * 3`|
|`/`|나눗셈|`10 / 3`|
|`//`|몫|`10 // 3`|
|`%`|나머지|`10 % 3`|
|`**`|거듭제곱|`10 ** 3`|

예시:

```python
a = 10
b = 3

print(a / b)    # 3.333...
print(a // b)   # 3
print(a % b)    # 1
print(a ** b)   # 1000
```

## 7. `//`와 `%`

`//`는 몫을 구하고, `%`는 나머지를 구한다.

```python
change = 750

coin500 = change // 500
change = change % 500

print(coin500)
print(change)
```

자판기 거스름돈 계산에서 자주 사용된다.

내 식으로 이해하면:

```text
// 는 몇 개 줄 수 있는지 계산
% 는 주고 남은 나머지를 계산
```

## 8. 할당 연산자

변수에 값을 넣는 연산자는 `=`이다.

```python
x = 10
```

여러 변수에 동시에 값을 넣을 수도 있다.

```python
x, y, z = 10, 20, 30
```

변수 값을 서로 바꿀 수도 있다.

```python
x, y = y, x
```

## 9. 복합 대입 연산자

기존 변수 값을 이용해서 다시 저장할 때 사용한다.

```python
x = 10

x += 1   # x = x + 1
x -= 1   # x = x - 1
x *= 2   # x = x * 2
x /= 2   # x = x / 2
```

반복문에서 카운트를 증가시킬 때 자주 사용한다.

```python
count = 0
count += 1
```

## 10. 연산자 우선순위

Python은 수학처럼 곱셈과 나눗셈을 덧셈과 뺄셈보다 먼저 계산한다.

```python
print(10 + 20 / 2)
```

결과는 `20.0`이다.

괄호를 사용하면 계산 순서를 바꿀 수 있다.

```python
print((10 + 20) / 2)
```

결과는 `15.0`이다.

## 11. 문자열

문자열은 글자들의 모음이다.

```python
name = "Kim"
message = "Hello Python"
```

문자열은 큰따옴표 또는 작은따옴표를 사용할 수 있다.

```python
a = "hello"
b = 'hello'
```

문자열끼리는 `+`로 연결할 수 있다.

```python
first = "Hello"
second = "Python"

print(first + " " + second)
```

문자열은 `*`로 반복할 수 있다.

```python
print("-" * 30)
```

## 12. 숫자와 문자열의 차이

```python
print(100 + 200)
print("100" + "200")
```

결과:

```text
300
100200
```

숫자는 계산되고, 문자열은 이어붙여진다.

## 13. 형변환

문자열을 숫자로 바꾸려면 `int()`나 `float()`를 사용한다.

```python
x = int("100")
y = float("3.14")
```

숫자를 문자열로 바꾸려면 `str()`을 사용한다.

```python
age = 21
print("나이: " + str(age))
```

## 14. `input()`

`input()`은 사용자에게 값을 입력받는다.

```python
name = input("이름을 입력하세요: ")
print(name)
```

중요한 점은 `input()`으로 받은 값은 항상 문자열이라는 것이다.

```python
x = input("첫 번째 수: ")
y = input("두 번째 수: ")

print(x + y)
```

만약 `10`, `20`을 입력하면 결과는 `30`이 아니라 `"1020"`이 된다.

숫자 계산을 하려면 형변환해야 한다.

```python
x = int(input("첫 번째 수: "))
y = int(input("두 번째 수: "))

print(x + y)
```

## 15. `print()`

`print()`는 화면에 값을 출력한다.

```python
print("Hello")
print(10 + 20)
```

변수와 문자열을 함께 출력할 수 있다.

```python
x = 100
y = 200

print(x, "와", y, "의 합 =", x + y)
```

f-string을 사용하면 더 깔끔하다.

```python
print(f"{x}와 {y}의 합 = {x + y}")
```

## 대표 예제: BMI 계산

```python
weight = float(input("몸무게를 kg 단위로 입력하시오: "))
height = float(input("키를 미터 단위로 입력하시오: "))

bmi = weight / (height ** 2)

print("당신의 BMI =", bmi)
```

## 대표 예제: 자판기 거스름돈 계산

```python
itemPrice = int(input("물건값을 입력하시오: "))
note = int(input("1000원 지폐개수: "))
coin500 = int(input("500원 동전개수: "))
coin100 = int(input("100원 동전개수: "))

change = note * 1000 + coin500 * 500 + coin100 * 100 - itemPrice

nCoin500 = change // 500
change = change % 500

nCoin100 = change // 100
change = change % 100

nCoin10 = change // 10
change = change % 10

nCoin1 = change

print("500원 =", nCoin500, "100원 =", nCoin100, "10원 =", nCoin10, "1원 =", nCoin1)
```

## 헷갈린 점 정리

### `=`와 `==`

|기호|의미|
|---|---|
|`=`|값을 대입|
|`==`|같은지 비교|

```python
x = 10

if x == 10:
    print("같다")
```

### `/`와 `//`

|기호|의미|
|---|---|
|`/`|나눗셈 결과|
|`//`|몫|

```python
print(10 / 3)
print(10 // 3)
```

### 숫자 100과 문자열 "100"

```python
100       # 숫자
"100"     # 문자열
```

계산하려면 숫자여야 한다.

## 내 식으로 이해

변수와 수식은 Python 프로그램의 가장 기본적인 재료이다.

전자회로에서 전압, 전류, 저항 같은 값을 변수로 두고 공식을 적용하듯이, Python에서도 입력값을 변수에 저장하고 수식으로 계산한 뒤 결과를 출력한다.

즉, Python 프로그램의 기본 흐름은 다음과 같다.

```text
입력 → 변수 저장 → 계산 → 출력
```
