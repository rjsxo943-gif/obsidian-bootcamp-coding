## 한 줄 정의

클래스는 객체를 만들기 위한 설계도이고, 객체는 그 설계도로 만든 실제 대상이다.

객체는 속성과 동작을 가진다.

```python
class Car:
    def __init__(self, speed, color, model):
        self.speed = speed
        self.color = color
        self.model = model
```

## 1. 객체 지향 프로그래밍

객체 지향 프로그래밍은 서로 관련 있는 데이터와 함수를 하나로 묶어서 객체로 만들고, 객체들이 서로 동작하면서 프로그램을 구성하는 방식이다.

절차 지향은 순서대로 실행되는 절차를 중심으로 생각한다.

객체 지향은 데이터와 동작을 가진 객체를 중심으로 생각한다.

## 2. 객체

객체는 속성과 동작을 가진다.

예를 들어 자동차 객체는 다음과 같은 속성을 가질 수 있다.

```text
색상
속도
모델
연식
가격
```

자동차 객체는 다음과 같은 동작을 가질 수 있다.

```text
주행한다
멈춘다
방향을 바꾼다
속도를 올린다
```

## 3. 클래스

클래스는 객체를 만들기 위한 설계도이다.

```python
class Car:
    pass
```

클래스 이름은 보통 대문자로 시작한다.

```python
class Television:
    pass
```

## 4. 인스턴스

클래스로부터 만들어진 실제 객체를 인스턴스라고 한다.

```python
class Counter:
    pass

a = Counter()
b = Counter()
```

`a`와 `b`는 `Counter` 클래스로 만든 서로 다른 인스턴스이다.

## 5. 속성과 메서드

객체의 데이터는 속성이라고 한다.  
객체의 동작은 메서드라고 한다.

```python
class Counter:
    def __init__(self):
        self.count = 0

    def increment(self):
        self.count += 1

a = Counter()
a.increment()

print(a.count)
```

여기서 `count`는 속성이고, `increment()`는 메서드이다.

## 6. 생성자 `__init__`

생성자는 객체가 만들어질 때 자동으로 실행되는 메서드이다.

```python
class Counter:
    def __init__(self):
        self.count = 0
```

객체의 초기값을 설정할 때 사용한다.

```python
a = Counter()
```

이 순간 `__init__()`이 자동으로 실행된다.

## 7. `self`

`self`는 현재 객체 자신을 의미한다.

```python
class Counter:
    def __init__(self):
        self.count = 0
```

`self.count`는 현재 객체가 가진 `count` 속성을 의미한다.

## 8. 생성자에 매개변수 넣기

```python
class Counter:
    def __init__(self, initValue=0):
        self.count = initValue

    def increment(self):
        self.count += 1

a = Counter(100)
b = Counter()

print(a.count)
print(b.count)
```

기본값을 사용하면 값을 넣지 않아도 객체를 만들 수 있다.

## 9. TV 클래스 예제

```python
class Television:
    def __init__(self, channel, volume, on):
        self.channel = channel
        self.volume = volume
        self.on = on

    def show(self):
        print(self.channel, self.volume, self.on)

    def setChannel(self, channel):
        self.channel = channel

    def getChannel(self):
        return self.channel

t = Television(9, 10, True)
t.show()

t.setChannel(11)
t.show()
```

## 10. Circle 클래스 예제

```python
import math

class Circle:
    def __init__(self, radius=0):
        self.radius = radius

    def getArea(self):
        return math.pi * self.radius * self.radius

    def getPerimeter(self):
        return 2 * math.pi * self.radius

c = Circle(10)

print("원의 면적", c.getArea())
print("원의 둘레", c.getPerimeter())
```

## 11. Car 클래스 예제

```python
class Car:
    def __init__(self, speed, color, model):
        self.speed = speed
        self.color = color
        self.model = model

    def drive(self):
        self.speed = 60

myCar = Car(0, "blue", "E-class")

print("자동차 객체를 생성하였습니다.")
print("자동차의 속도는", myCar.speed)
print("자동차의 색상은", myCar.color)
print("자동차의 모델은", myCar.model)

myCar.drive()

print("자동차의 속도는", myCar.speed)
```

## 12. 캡슐화

캡슐화는 객체 내부의 세부 구현을 숨기고, 외부에는 필요한 기능만 제공하는 것이다.

예를 들어 TV 리모컨을 사용할 때 내부 회로를 몰라도 채널 변경 버튼만 누르면 동작한다.

프로그래밍에서도 객체 내부 데이터를 직접 건드리지 않고, 메서드를 통해 접근하게 만들 수 있다.

## 13. 정보 은닉

Python에서 변수 이름 앞에 `__`를 붙이면 외부에서 직접 접근하기 어렵게 만들 수 있다.

```python
class Student:
    def __init__(self, name=None, age=0):
        self.__name = name
        self.__age = age

obj = Student("Kim", 20)

print(obj.__age)
```

이 코드는 오류가 난다.

## 14. 접근자와 설정자

private 변수에 접근하려면 메서드를 만든다.

```python
class Student:
    def __init__(self, name=None, age=0):
        self.__name = name
        self.__age = age

    def getAge(self):
        return self.__age

    def getName(self):
        return self.__name

    def setAge(self, age):
        self.__age = age

    def setName(self, name):
        self.__name = name

obj = Student("Hong", 20)

print(obj.getName())
```

|종류|의미|
|---|---|
|getter|값을 읽는 메서드|
|setter|값을 변경하는 메서드|

## 15. 은행 계좌 클래스

```python
class BankAccount:
    def __init__(self):
        self.__balance = 0

    def withdraw(self, amount):
        self.__balance -= amount
        print("통장에서", amount, "가 출금되었음")
        return self.__balance

    def deposit(self, amount):
        self.__balance += amount
        print("통장에", amount, "가 입금되었음")
        return self.__balance

a = BankAccount()

a.deposit(100)
a.withdraw(10)
```

잔액 `__balance`는 외부에서 직접 접근하지 못하게 숨겼다.

## 16. 객체 참조

Python 변수는 객체 자체를 저장하는 것이 아니라 객체를 가리키는 참조를 저장한다.

```python
t = Television(11, 10, True)
s = t
```

이 경우 `s`와 `t`는 같은 객체를 가리킨다.

```python
s.channel = 9
t.show()
```

`s`로 바꿔도 `t`에서 바뀐 값이 보인다.

## 17. `is`

두 변수가 같은 객체를 가리키는지 확인할 때 사용한다.

```python
if s is t:
    print("2개의 변수는 동일한 객체를 참조하고 있습니다.")
```

다른 객체인지 확인하려면 `is not`을 사용한다.

```python
if s is not t:
    print("2개의 변수는 다른 객체를 참조하고 있습니다.")
```

## 18. `None`

변수가 아무 객체도 가리키지 않도록 하려면 `None`을 사용한다.

```python
myTV = None

if myTV is None:
    print("현재 TV가 없습니다.")
```

## 19. 객체를 함수로 전달

객체를 함수로 전달하면 객체의 내용이 바뀔 수 있다.

```python
class Television:
    def __init__(self, channel, volume, on):
        self.channel = channel
        self.volume = volume
        self.on = on

    def show(self):
        print(self.channel, self.volume, self.on)

def setSilentMode(t):
    t.volume = 2

myTV = Television(11, 10, True)

setSilentMode(myTV)
myTV.show()
```

결과:

```text
11 2 True
```

## 20. 클래스 변수

클래스 변수는 모든 객체가 공유하는 변수이다.

```python
class Television:
    serialNumber = 0

    def __init__(self, channel, volume, on):
        self.channel = channel
        self.volume = volume
        self.on = on

        Television.serialNumber += 1
        self.number = Television.serialNumber

    def show(self):
        print(self.channel, self.volume, self.on, self.number)

myTV = Television(11, 10, True)
myTV.show()
```

인스턴스 변수는 객체마다 따로 있고, 클래스 변수는 클래스 전체가 공유한다.

## 21. 인스턴스 변수와 클래스 변수

|구분|의미|
|---|---|
|인스턴스 변수|객체마다 따로 존재|
|클래스 변수|모든 객체가 공유|

예시:

```python
class Dog:
    kind = "Bulldog"

    def __init__(self, name, age):
        self.name = name
        self.age = age
```

`kind`는 클래스 변수이고, `name`, `age`는 인스턴스 변수이다.

## 22. 특수 메서드

Python에는 연산자와 연결된 특수 메서드가 있다.

예를 들어 `+`, `-`, `==`, `str()` 같은 동작을 클래스 안에서 정의할 수 있다.

```python
class Vector2D:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        return Vector2D(self.x + other.x, self.y + other.y)

    def __sub__(self, other):
        return Vector2D(self.x - other.x, self.y - other.y)

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

    def __str__(self):
        return f"({self.x}, {self.y})"

u = Vector2D(0, 1)
v = Vector2D(1, 0)

a = u + v

print(a)
```

## 내 식으로 이해

클래스는 설계도이고 객체는 설계도로 만든 실제 물건이다.

전자공학에서 회로 설계도가 있고, 그 설계도로 실제 보드를 만드는 것처럼, Python에서는 클래스를 정의하고 그 클래스로 객체를 만든다.

객체지향은 변수와 함수를 따로따로 두는 것이 아니라, 관련 있는 데이터와 동작을 하나의 단위로 묶어서 관리하는 방식이다.
