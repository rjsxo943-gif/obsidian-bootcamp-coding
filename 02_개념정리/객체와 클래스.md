# 객체와 클래스

> `객체와 클래스`, `__init__`, `self`, 인스턴스 변수, 클래스 변수, 특수 메소드, 상속, 다형성을 여기서 한 번에 복습한다.

---

## 1. 한 줄 정의

```text
클래스 = 객체를 만들기 위한 설계도
객체 = 클래스로 실제 만들어진 결과물
```

객체지향은 **데이터와 기능을 하나의 덩어리로 묶는 방식**이다.

예를 들어 TV는 채널, 볼륨, 전원 상태라는 데이터를 가지고 있고, 채널 변경이나 볼륨 변경 같은 동작을 가진다. 이런 것을 코드로 묶으면 클래스가 된다.

---

## 2. 기본 형태

```python
class Television:
    def __init__(self, channel, volume, on):
        self.channel = channel
        self.volume = volume
        self.on = on
```

```python
tv1 = Television(1, 10, True)
tv2 = Television(2, 20, False)
```

`Television`은 클래스이고, `tv1`, `tv2`는 객체이다. 같은 설계도로 만들었지만 서로 다른 값을 가질 수 있다.

---

## 3. `__init__`과 `self`

`__init__`은 객체가 생성될 때 자동으로 실행되는 메소드이다.

```python
def __init__(self, channel, volume, on):
    self.channel = channel
    self.volume = volume
    self.on = on
```

`self`는 현재 객체 자신을 의미한다.

```text
self.channel = 이 객체의 channel 값
self.volume = 이 객체의 volume 값
self.on = 이 객체의 on 값
```

즉 `tv1.channel`과 `tv2.channel`이 서로 다를 수 있는 이유가 `self` 때문이다.

---

## 4. 인스턴스 변수와 클래스 변수

### 인스턴스 변수

객체마다 따로 가지는 값이다.

```python
self.channel = channel
self.volume = volume
self.on = on
```

`tv1`과 `tv2`는 같은 클래스에서 만들어졌지만 채널과 볼륨이 다를 수 있다.

### 클래스 변수

같은 클래스로 만든 객체들이 공유하는 값이다.

```python
class Television:
    serial_number = 0

    def __init__(self, channel, volume, on):
        Television.serial_number += 1
        self.serial_number = Television.serial_number
```

객체가 생성될 때마다 `Television.serial_number`가 증가하므로, TV 객체 번호처럼 사용할 수 있다.

|구분|인스턴스 변수|클래스 변수|
|---|---|---|
|위치|`self.변수명`|클래스 바로 아래|
|공유 여부|객체마다 따로|모든 객체가 공유|
|예시|`self.channel`|`Television.serial_number`|

## 5. 메소드

클래스 안에 정의된 함수를 메소드라고 한다.

```python
def set_channel(self, channel):
    self.channel = channel

def get_channel(self):
    return self.channel
```

메소드는 객체가 가진 데이터를 사용하거나 바꾼다.

```python
tv1.set_channel(7)
print(tv1.get_channel())
```

## 6. 특수 메소드

특수 메소드는 Python이 특정 문법을 만났을 때 자동으로 호출하는 메소드이다.

### `__str__`

객체를 출력할 때 사용할 문자열을 정한다.

```python
def __str__(self):
    return f"Television(channel={self.channel}, volume={self.volume}, on={self.on})"
```

```python
print(tv1)
```

`__str__`이 있으면 객체가 보기 좋은 문자열로 출력된다.

### `__add__`

객체끼리 `+` 연산을 할 때 호출된다.

```python
def __add__(self, other):
    new_width = self.width + other.width
    new_height = self.height + other.height
    return Rectangle(new_width, new_height)
```

```python
rect3 = rect1 + rect2
```

실제로는 `rect1.__add__(rect2)`가 실행되는 느낌이다.

## 7. 상속

상속은 기존 클래스의 속성과 기능을 물려받아 새로운 클래스를 만드는 것이다.

```python
class ElectricCar(Car):
    def __init__(self, make, model, color, price, battery_capacity):
        super().__init__(make, model, color, price)
        self.battery_capacity = battery_capacity
```

`ElectricCar`는 `Car`를 상속받는다. 즉 전기차는 자동차의 기본 속성을 가지면서 배터리 용량이라는 속성을 추가로 가진다.

`super().__init__()`은 부모 클래스의 생성자를 호출하는 코드이다.

## 8. 다형성

다형성은 같은 이름의 메소드를 호출해도 객체 종류에 따라 다르게 동작하는 것이다.

```python
class Shape:
    def area(self):
        raise NotImplementedError
```

```python
class Circle(Shape):
    def area(self):
        return math.pi * self.radius ** 2
```

```python
class Rectangle(Shape):
    def area(self):
        return self.width * self.height
```

```python
for shape in shapeList:
    print(shape.area())
```

같은 `area()`를 호출해도 원이면 원의 넓이, 사각형이면 사각형의 넓이를 계산한다.

## 9. 내 식으로 이해

객체지향은 어려운 문법이라기보다 **현실의 대상을 코드 안에서 하나의 덩어리로 만드는 방법**이다.

```text
TV = 채널 + 볼륨 + 전원상태 + 채널변경 기능
자동차 = 속도 + 색상 + 모델 + 주행 기능
원 = 반지름 + 넓이계산 + 둘레계산
```

즉 변수만 따로 두는 게 아니라, 그 변수가 해야 하는 동작까지 같이 묶는 것이다.
