# Obsidian Bootcamp Coding Notes

부트캠프 코딩 학습 내용을 정리하는 Obsidian Vault입니다.

이 저장소는 코드를 보관하는 곳이 아니라, 수업에서 배운 내용과 GitHub 코드 해설을 연결해두는 학습 노트 저장소입니다.

---

## 목적

```text
날짜별 학습 기록
→ 코드 해설
→ 개념 정리
→ 에러노트
→ 프로젝트 정리
```

수업 중 작성한 코드는 `mycodehouse` 저장소에 두고, 이 저장소에는 그 코드를 이해하기 위한 Markdown 노트를 둡니다.

---

## 폴더 구조

```text
00_인덱스/
01_날짜별_학습기록/
02_개념정리/
03_코드해설/
04_에러노트/
05_프로젝트정리/
99_템플릿/
```

|폴더|역할|
|---|---|
|`00_인덱스`|전체 노트로 이동하는 메인 허브|
|`01_날짜별_학습기록`|날짜별 학습 기록|
|`02_개념정리`|큰 주제별 개념 복습 노트|
|`03_코드해설`|GitHub 코드에 대한 날짜별 통합 해설|
|`04_에러노트`|자주 만나는 오류와 해결 방법|
|`05_프로젝트정리`|완성물/미니 프로젝트 중심 정리|
|`99_템플릿`|반복해서 쓰는 노트 양식|

---

## 운영 기준

개념 노트는 너무 잘게 쪼개지 않고, 복습할 때 한 번에 읽을 수 있는 단위로 유지합니다.

예를 들어 0706 객체지향 수업은 다음처럼 정리합니다.

```text
객체와 클래스
├─ 클래스와 객체
├─ self와 __init__
├─ 인스턴스 변수와 클래스 변수
├─ 특수 메소드
├─ 상속
└─ 다형성
```

GUI 관련 내용도 다음처럼 하나의 큰 노트로 합칩니다.

```text
tkinter GUI 기초
├─ GUI 기본 구조
├─ 이벤트와 콜백
├─ 파일 열기/저장
└─ Canvas와 마우스 이벤트
```
