## 📚 배운 개념 요약

### 1. 객체지향

객체지향은 데이터와 기능을 하나로 묶는 방식이다.

```python
class Television:
    def __init__(self, channel, volume, on):
        self.channel = channel
        self.volume = volume
        self.on = on
```

오늘은 `Television`, `Circle`, `Car`, `Rectangle`, `Dice` 같은 클래스를 만들면서 객체, 생성자, 메소드, 특수 메소드, 상속, 다형성을 실습했다.

자세한 개념은 [[객체와 클래스]]에서 정리한다.

---

### 2. tkinter GUI

`tkinter`는 창이 있는 프로그램을 만들기 위한 Python 기본 GUI 도구이다.

```python
root = tk.Tk()
root.title("메모장")
root.geometry("800x600")
root.mainloop()
```

메모장에서는 `Text`, `Menu`, 파일 열기/저장을 사용했고, 그림판에서는 `Canvas`, 마우스 이벤트, 색상 선택, 지우개 기능을 사용했다.

자세한 개념은 [[tkinter GUI 기초]]에서 정리한다.

---

### 3. NumPy 배열

Python 리스트를 NumPy 배열로 바꾸면 수치 계산에 더 적합한 자료구조가 된다.

```python
ftemp = [32, 63, 73, 82, 91]
F = np.array(ftemp)
```

자세한 개념은 [[NumPy 배열 기초]]에서 정리한다.

## 🧠 내 식으로 이해

오늘 배운 내용은 결국 **객체지향으로 프로그램을 구조화하고, 그 구조를 이용해서 실제 창이 있는 프로그램까지 만들어 본 것**이다.

`class`는 단순한 문법이 아니라 TV, 자동차, 원, 사각형 같은 대상을 코드 안에서 하나의 덩어리로 만드는 방식이다.

`tkinter`는 그 덩어리를 실제 사용자가 조작할 수 있는 창 프로그램으로 보여주는 연습이었다.
