## 1. `while` 반복문

### 핵심 코드

```python
treeHit = 0

while treeHit < 10:
    treeHit += 1
    print("나무를 %d번 찍었습니다." % treeHit)

    if treeHit == 10:
        print("나무 넘어갑니다.")

print("나무를 %d번 찍었습니다." % treeHit)
```

### 코드 목적

나무를 10번 찍을 때까지 반복하고, 10번이 되면 나무가 넘어간다는 메시지를 출력하는 코드이다.

---

### 배운 개념 1: `while`

```python
while treeHit < 10:
```

`while`은 조건이 참인 동안 계속 반복한다.

이 조건은 `treeHit`이 10보다 작을 때 반복하라는 뜻이다.

---

### 배운 개념 2: 반복 횟수 증가

```python
treeHit += 1
```

이 코드는 다음과 같은 의미이다.

```python
treeHit = treeHit + 1
```

즉, 반복할 때마다 `treeHit` 값이 1씩 증가한다.

---

### 주의할 점: 들여쓰기

마지막 `print()`는 `while`문 밖에 있다.

```python
print("나무를 %d번 찍었습니다." % treeHit)
```

그래서 반복문이 끝난 뒤 한 번 더 실행된다.

Python에서는 들여쓰기에 따라 코드가 반복문 안에 있는지 밖에 있는지가 결정된다.

---

## 2. `break`

### 핵심 코드

```python
if coffee == 0:
    print("커피가 다 떨어졌습니다. 판매를 중지 합니다.")
    break
```

### 코드 목적

반복문을 진행하다가 특정 조건이 만족되면 반복을 강제로 종료한다.

`break`는 반복문 자체를 끝내는 명령이다.

---

### 내 식으로 이해

`break`는 PLC에서 특정 조건이 만족되면 동작을 강제로 멈추거나, 다음 공정으로 넘어가는 것과 비슷하다.

예를 들어 커피 자판기 코드에서는 커피가 0개가 되는 순간 판매를 멈춰야 하므로 `break`를 사용한다.

---

## 3. `continue`

### 핵심 코드

```python
a = 0

while a < 10:
    a = a + 1

    if a % 2 == 0:
        continue

    print(a)
```

### 코드 목적

1부터 10까지 숫자 중 홀수만 출력한다.

---

### 코드 해설

```python
if a % 2 == 0:
    continue
```

`a % 2 == 0`은 짝수라는 뜻이다.

짝수일 때 `continue`를 만나면 아래의 `print(a)`를 실행하지 않고 다음 반복으로 넘어간다.

따라서 출력 결과는 홀수만 나온다.

```text
1
3
5
7
9
```

---

### `break`와 `continue` 차이

|키워드|의미|
|---|---|
|`break`|반복문 자체를 종료|
|`continue`|이번 반복만 건너뛰고 다음 반복으로 이동|

## 4. `for` 반복문과 `range()`

### 핵심 코드

```python
s = 0

for i in range(1, 101):
    s += i

print(s)
```

### 코드 목적

1부터 100까지의 합을 구한다.

---

### 코드 해설

```python
range(1, 101)
```

`range(1, 101)`은 1부터 100까지의 숫자를 만든다.

끝값인 `101`은 포함되지 않는다.

```python
s += i
```

이 코드는 현재까지의 합계에 `i`를 계속 더하는 코드이다.

결과는 다음과 같다.

```text
5050
```

## 5. 조건과 반복 같이 쓰기

### 홀수의 합

```python
s = 0

for i in range(1, 101):
    if i % 2 == 1:
        s += i

print(s)
```

### 코드 해설

`%`는 나머지를 구하는 연산자이다.

```python
i % 2 == 1
```

이 조건은 2로 나누었을 때 나머지가 1인 수를 의미한다.

즉, 홀수만 골라서 더하는 코드이다.

---

### 6의 배수의 합

```python
s = 0

for i in range(1, 101):
    if i % 6 == 0:
        s += i

print(s)
```

`i % 6 == 0`은 6으로 나누어떨어지는 수를 의미한다.

즉, 6의 배수만 누적해서 더한다.

## 6. 중첩 반복문

### 구구단 예제

```python
i = 2

while i <= 3:
    j = 1

    while j <= 9:
        print(f"{i:3} x {j:3} = {i * j:3}")
        j += 1

    i += 1
```

### 코드 목적

2단과 3단 구구단을 출력한다.

---

### 코드 해설

반복문 안에 반복문이 들어간 구조를 중첩 반복문이라고 한다.

|반복문|역할|
|---|---|
|바깥 반복문|몇 단인지 결정|
|안쪽 반복문|1부터 9까지 곱함|

즉, 구구단처럼 같은 구조가 반복되는 문제에 적합하다.

## 7. 별 찍기 패턴

### 정삼각형 예제

```python
n = 4

for i in range(1, n + 1):
    for j in range(n - i):
        print(" ", end=" ")

    for j in range(2 * i - 1):
        print("*", end=" ")

    print()
```

### 코드 목적

반복문을 사용해서 정삼각형 모양의 별을 출력한다.

---

### 핵심 공식

|모양|공백 개수|별 개수|
|---|---|---|
|직각삼각형|없음|`i`|
|오른쪽 정렬 삼각형|`n - i`|`i`|
|정삼각형|`n - i`|`2 * i - 1`|
|마름모 위쪽|`n - i`|`2 * i - 1`|
|마름모 아래쪽|`n - i`|`2 * i - 1`, 단 `i` 감소|

별 찍기는 단순히 별을 출력하는 문제가 아니라, 반복문의 구조와 인덱스 변화를 눈으로 확인하는 연습이다.

## 9. 함수 기반 계산기

### 핵심 코드

```python
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return "Error: Division by zero"

    return a / b
```

### 코드 목적

사칙연산을 각각 함수로 나누어 계산기 구조를 만든다.

---

### 함수별 역할

|함수|역할|
|---|---|
|`add()`|덧셈|
|`subtract()`|뺄셈|
|`multiply()`|곱셈|
|`divide()`|나눗셈|

---

### 나눗셈 예외 처리

```python
if b == 0:
    return "Error: Division by zero"
```

0으로 나누는 것은 불가능하다.

그래서 나눗셈을 하기 전에 `b`가 0인지 먼저 확인한다.

## 10. 명령행 인자 `sys.argv`

### `sys1.py` 핵심 코드

```python
import sys

args = sys.argv[1:]

for i in args:
    print(i)
```

### 코드 목적

터미널에서 Python 파일을 실행할 때 같이 입력한 값을 코드 안에서 받는다.

---

### 실행 예시

```powershell
python sys1.py hello python 123
```

이렇게 실행하면 `hello`, `python`, `123`이 프로그램 안으로 들어온다.

`sys.argv`는 다음과 비슷한 리스트가 된다.

```python
["sys1.py", "hello", "python", "123"]
```

---

### `sys.argv[0]`

```python
sys.argv[0]
```

실행한 파일 이름이다.

---

### `sys.argv[1:]`

```python
sys.argv[1:]
```

파일 이름을 제외한 나머지 입력값 전체이다.

즉, 실제 사용자가 입력한 값만 가져오려면 `sys.argv[1:]`를 사용한다.

## 14. 명령행 인자 평균 계산

### `avg_args.py` 핵심 코드

```python
import sys

args = sys.argv[1:]

if len(args) == 0:
    print("숫자를 하나 이상 입력하세요.")
else:
    total = 0

    for i in args:
        total += int(i)

    avg = total / len(args)

    print("합계:", total)
    print("평균:", f"{avg:.2f}")
```

### 코드 목적

명령행 인자로 받은 숫자들의 합계와 평균을 계산한다.

---

### 입력값이 없을 때 처리

```python
if len(args) == 0:
    print("숫자를 하나 이상 입력하세요.")
```

입력값이 하나도 없는데 평균을 계산하면 `0`으로 나누게 된다.

그래서 먼저 입력값의 개수를 확인한다.

---

### 평균 계산

```python
avg = total / len(args)
```

합계를 입력값 개수로 나누면 평균이다.

---

### 소수점 둘째 자리 출력

```python
f"{avg:.2f}"
```

`:.2f`는 소수점 둘째 자리까지 출력하라는 뜻이다.

## ✅ 오늘의 핵심 정리

```text
while은 조건이 참인 동안 반복한다.
for는 정해진 범위나 리스트를 하나씩 순회한다.
range(1, 101)은 1부터 100까지를 의미한다.
break는 반복문을 종료한다.
continue는 이번 반복을 건너뛰고 다음 반복으로 간다.
중첩 반복문은 반복문 안에 반복문이 들어간 구조이다.
함수는 코드를 기능 단위로 묶는 방법이다.
return은 함수 결과를 밖으로 돌려준다.
print()는 화면에 보여주는 것이고, return은 값을 돌려주는 것이다.
sys.argv는 터미널 입력값을 Python 코드에서 받는 방법이다.
sys.argv로 받은 값은 문자열이므로 숫자 계산에는 int() 변환이 필요하다.
파일 입출력에서는 open(), close(), encoding 개념이 중요하다.
```

## ⚠️ 오늘 헷갈릴 수 있는 부분

## 1. `while`과 `for`의 차이

|구분|설명|
|---|---|
|`while`|조건이 참인 동안 반복|
|`for`|정해진 범위나 자료를 순서대로 반복|

---

## 2. `break`와 `continue`의 차이

|코드|의미|
|---|---|
|`break`|반복문 자체를 끝냄|
|`continue`|이번 반복만 건너뛰고 다음 반복으로 감|

---

## 3. `return`과 `print()`의 차이

|코드|의미|
|---|---|
|`print()`|화면에 보여줌|
|`return`|함수 밖으로 값을 돌려줌|

---

## 4. `sys.argv[1:]`가 필요한 이유

`sys.argv[0]`은 실행한 파일 이름이다.

그래서 실제 사용자가 입력한 값만 가져오려면 다음처럼 쓴다.

```python
args = sys.argv[1:]
```

오늘 배운 내용은 Python 프로그램이 단순 계산을 넘어서, 흐름을 제어하고 기능을 분리하고 외부 입력을 받는 단계로 넘어간 것이다.

이전에는 `input()`으로 프로그램 실행 중에 값을 입력받았다면, 오늘은 `sys.argv`를 통해 터미널 실행 시점에 값을 넘기는 방식도 배웠다.
