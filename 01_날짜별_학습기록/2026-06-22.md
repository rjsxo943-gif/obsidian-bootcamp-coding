## 📚 배운 개념

## 1. `print()` 함수

`print()`는 화면에 값을 출력하는 함수이다.

문자열을 출력할 수도 있고, 계산 결과를 함께 출력할 수도 있다.

```python
print("Welcome to Python")
```

이 코드는 화면에 다음 문장을 출력한다.

```text
Welcome to Python
```

---

## 2. 문자열 출력

문자열은 큰따옴표 `" "` 또는 작은따옴표 `' '` 안에 적는다.

```python
print("원하는 문자열을 출력할 때는 print함수를 이용함")
```

이 코드는 사용자가 원하는 문장을 그대로 화면에 보여준다.

---

## 3. 계산 결과 출력

```python
print("1 + 2 =", 1 + 2)
```

이 코드는 문자열 `"1 + 2 ="`와 실제 계산 결과 `3`을 함께 출력한다.

결과는 다음과 같다.

```text
1 + 2 = 3
```

즉, Python은 단순히 글자만 출력하는 것이 아니라 산술 계산도 바로 수행할 수 있다.

---

## 🧠 내 식으로 이해

오늘 배운 내용은 Python 프로그램의 가장 기본적인 입출력 시작점이다.

프로그램은 결국 사용자가 원하는 값을 계산하거나 처리해서 화면에 보여주는 것인데, 그 첫 단계가 `print()`이다.

전자공학이나 PLC에서 출력 장치에 신호를 보내는 것처럼, Python에서는 `print()`가 화면으로 값을 내보내는 가장 기본적인 출력 명령이라고 볼 수 있다.
