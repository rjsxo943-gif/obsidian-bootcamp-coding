## 2. TTS 실습

TTS는 Text To Speech의 약자로, 텍스트를 음성으로 바꾸는 기능이다.

오늘은 `gTTS`를 이용해 문자열을 `.mp3` 파일로 저장하는 실습을 했다.

```python
from gtts import gTTS

text = "오늘 배운 내용을 음성으로 저장합니다."
tts = gTTS(text=text, lang="ko")
tts.save("news_Son.mp3")
```

### 핵심

```text
텍스트 입력
→ gTTS 객체 생성
→ mp3 파일 저장
```

이 실습을 통해 Python 코드가 화면 출력뿐 아니라 실제 파일을 생성할 수 있다는 것을 확인했다.

관련 해설: [[0707_코드해설#1. gTTS로 음성 파일 만들기]]

## 5. Series와 DataFrame

간단한 리스트를 이용해 Series를 만들었다.

```python
data = ['Kim', 'Park', 'Lee', 'Choi']
```

딕셔너리 형태의 데이터를 이용해 DataFrame도 만들었다.

## 6. Iris 머신러닝 실습

`ml.ipynb`에서는 scikit-learn의 Iris 데이터셋을 사용했다.

```python
from sklearn.datasets import load_iris

iris = load_iris()
```

Iris 데이터셋은 붓꽃 품종을 분류하는 대표적인 머신러닝 예제 데이터이다.

### Iris 데이터 구성

| 항목 | 의미 |
|---|---|
| `iris.data` | 입력 데이터 |
| `iris.target` | 정답 라벨 |
| `iris.target_names` | 품종 이름 |
| `iris.feature_names` | 특성 이름 |

입력 특성은 다음 4개이다.

```text
sepal length
sepal width
petal length
petal width
```

분류 대상은 다음 3개이다.

```text
setosa
versicolor
virginica
```

관련 개념: [[0707_개념정리#14. Iris 데이터셋]]

## 7. SVM 모델 학습과 예측

SVM 모델을 생성하고 Iris 데이터로 학습했다.

```python
from sklearn import svm

s = svm.SVC(gamma=0.1, C=10)
s.fit(iris.data, iris.target)
```

새로운 샘플을 넣어 예측했다.

```python
new_d = [
    [6.4, 3.2, 6.0, 2.5],
    [7.1, 3.1, 4.7, 1.35]
]

res = s.predict(new_d)
```

출력 결과:

```text
새로운 2개 샘플의 부류 번호는 [2 1]
새로운 2개 샘플의 부류 이름은 ['virginica' 'versicolor']
```

### 오늘 이해한 머신러닝 흐름

```text
데이터 준비
→ 모델 생성
→ fit()으로 학습
→ predict()로 예측
→ 결과 해석
```

관련 개념: [[0707_개념정리#16. SVM]], [[0707_개념정리#17. fit]], [[0707_개념정리#18. predict]]

## 1. pandas와 plotly 역할 혼동

잘못된 코드:

```python
df = pd.data.iris()
fig = pd.scatter_3d(...)
```

`pandas`에는 `data.iris()`도 없고 `scatter_3d()`도 없다.

올바른 역할 분리:

```text
데이터셋 로드 → sklearn.datasets
데이터 정리 → pandas
머신러닝 → sklearn
시각화 → plotly.express
```

> [!bug]
> `pd.scatter_3d()`가 아니라 `px.scatter_3d()`를 써야 한다.
