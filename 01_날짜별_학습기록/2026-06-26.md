## 1. `stringEx.py` 코드 해설

### 핵심 코드

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

print(f"a + b = {a + b}")
print(f"a - b = {a - b}")
print(f"a * b = {a * b}")
print(f"a / b = {a / b}")
```

### 코드 목적

사용자에게 숫자 2개를 입력받고, 두 숫자에 대해 사칙연산 결과를 출력하는 코드이다.

---

### 배운 개념 1: `input()`

`input()`은 사용자에게 값을 입력받는 함수이다.

```python
a = input("Enter first number: ")
```

하지만 `input()`으로 받은 값은 기본적으로 문자열이다.

예를 들어 사용자가 `10`을 입력해도 Python 내부에서는 숫자 `10`이 아니라 문자열 `"10"`으로 저장된다.

---

### 배운 개념 2: `int()` 형변환

숫자 계산을 하려면 입력받은 문자열을 정수로 바꿔야 한다.

```python
a = int(input("Enter first number: "))
```

이 코드는 사용자가 입력한 값을 정수로 변환해서 `a`에 저장한다.

---

### 배운 개념 3: f-string

```python
print(f"a + b = {a + b}")
```

`f""` 안에서 `{}`를 사용하면 변수나 계산식을 문자열 안에 바로 넣을 수 있다.

예를 들어 `a = 10`, `b = 3`이면 다음처럼 출력된다.

```text
a + b = 13
```

---

### 사칙연산 정리

|코드|의미|
|---|---|
|`a + b`|덧셈|
|`a - b`|뺄셈|
|`a * b`|곱셈|
|`a / b`|나눗셈|

## 2. `1.py` 코드 해설

### 핵심 코드

```python
y = []

for i in range(5):
    a = int(input("5개의 정수값을 입력하시오: "))
    y.append(a)

c = len(y)

print(y)
print(f"mean = {sum(y) / c}, sum = {sum(y)}, min = {min(y)}, max = {max(y)}")
```

### 코드 목적

정수 5개를 입력받아서 리스트에 저장하고, 리스트의 합계, 평균, 최솟값, 최댓값을 계산하는 코드이다.

---

### 배운 개념 1: 빈 리스트 생성

```python
y = []
```

`[]`는 빈 리스트를 만든다.

리스트는 여러 값을 순서대로 저장할 수 있는 자료구조이다.

---

### 배운 개념 2: 반복 입력

```python
for i in range(5):
```

`range(5)`는 0부터 4까지 총 5개의 값을 만든다.

즉, `for`문이 5번 반복된다.

이 구조를 사용하면 사용자에게 값을 5번 입력받을 수 있다.

---

### 배운 개념 3: 리스트에 값 추가

```python
y.append(a)
```

`append()`는 리스트의 맨 뒤에 값을 추가한다.

예를 들어 사용자가 `10`, `20`, `30`, `40`, `50`을 입력하면 리스트는 다음처럼 된다.

```python
[10, 20, 30, 40, 50]
```

---

### 배운 개념 4: 리스트 통계 함수

```python
sum(y)
min(y)
max(y)
len(y)
```

|함수|의미|
|---|---|
|`sum(y)`|리스트 요소의 합계|
|`min(y)`|리스트 요소 중 최솟값|
|`max(y)`|리스트 요소 중 최댓값|
|`len(y)`|리스트의 길이|

평균은 다음처럼 계산한다.

```python
avg = sum(y) / len(y)
```

---

### 개선 포인트

원래 코드에서 `c = int(len(y))`라고 작성했지만, `len(y)`는 이미 정수이므로 `int()`를 다시 사용할 필요는 없다.

```python
c = len(y)
```

이렇게 쓰면 더 깔끔하다.

## 3. `2.py` 코드 해설

### 핵심 코드

```python
fruits = []

for i in range(5):
    fruit = input("enter a fruit: ")
    fruits.append(fruit)

print(fruits)
print(fruits[-1])
print(fruits[3])
```

### 코드 목적

문자열 5개를 입력받아 리스트에 저장하고, 리스트 전체와 특정 위치의 값을 출력하는 코드이다.

---

### 배운 개념 1: 문자열 리스트 입력

```python
fruit = input("enter a fruit: ")
fruits.append(fruit)
```

`input()`으로 입력받은 과일 이름은 문자열이다.

입력받은 문자열을 `append()`로 리스트에 추가한다.

예를 들어 다음처럼 입력했다고 하자.

```text
apple
banana
melon
grape
orange
```

리스트는 다음처럼 저장된다.

```python
["apple", "banana", "melon", "grape", "orange"]
```

---

### 배운 개념 2: 리스트 인덱싱

Python 리스트의 인덱스는 0부터 시작한다.

```python
fruits[0]
```

첫 번째 요소를 의미한다.

```python
fruits[3]
```

네 번째 요소를 의미한다.

```python
fruits[-1]
```

마지막 요소를 의미한다.

---

### 인덱스 예시

```python
fruits = ["apple", "banana", "melon", "grape", "orange"]
```

|코드|결과|의미|
|---|---|---|
|`fruits[0]`|`"apple"`|첫 번째 요소|
|`fruits[1]`|`"banana"`|두 번째 요소|
|`fruits[3]`|`"grape"`|네 번째 요소|
|`fruits[-1]`|`"orange"`|마지막 요소|
